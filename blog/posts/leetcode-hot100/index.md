# 📘 LeetCode Hot100

这部分现在按 **题型专题** 来组织，本质上就是一个可以连续阅读的系列：从数组与哈希开始，逐步走到二分、DP、贪心与设计题。

目标很明确：

- 100 题完整覆盖
- 每题都有思路
- 每题都有 JavaScript 代码
- 更适合前端面试前按类型复习

## 这个系列怎么组织

- 按题型拆成 12 篇，方便顺着刷
- 目标是把 Hot100 里常见套路真正串起来，而不是只列题名
- 每一篇都先讲“怎么识别这一类题”，再给题目、思路、代码

## 适合怎么刷

- **第一次系统刷**：按目录顺序走
- **面试前突击**：优先看数组哈希、滑窗、链表、二叉树、DP
- **刷过但容易忘**：按题型把模板重新过一遍
