# 07 - 二叉树进阶与图论

这一组是树和图里的高频进阶题。核心不是遍历本身，而是 **如何在遍历中维护额外状态**。

## 这一类题怎么识别

- 题目要求构造树、找祖先、统计路径数
- 图里涉及依赖关系、可达性、层次扩散
- 通常需要 DFS / BFS + 额外状态

## 这类题的通用思路

树题先想递归返回值；图题先想邻接表、入度表或 visited 状态。

## 51. 二叉搜索树中第 K 小的元素

- **标签**：BST / 中序遍历
- **复杂度**：时间 O(h+k)，空间 O(h)

### 思路

BST 中序遍历天然有序，第 k 次访问到的节点就是答案。

### 代码（JavaScript）

```js
function kthSmallest(root, k) {
  const stack = [];
  let cur = root;
  while (cur || stack.length) {
    while (cur) {
      stack.push(cur);
      cur = cur.left;
    }
    cur = stack.pop();
    if (--k === 0) return cur.val;
    cur = cur.right;
  }
}
```

## 52. 从前序与中序遍历序列构造二叉树

- **标签**：二叉树 / 分治
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

前序首元素定根，中序里根左边是左子树，右边是右子树。

### 代码（JavaScript）

```js
function buildTree(preorder, inorder) {
  const indexMap = new Map();
  inorder.forEach((val, i) => indexMap.set(val, i));
  function dfs(pl, pr, il, ir) {
    if (pl > pr) return null;
    const rootVal = preorder[pl];
    const idx = indexMap.get(rootVal);
    const leftSize = idx - il;
    const root = new TreeNode(rootVal);
    root.left = dfs(pl + 1, pl + leftSize, il, idx - 1);
    root.right = dfs(pl + leftSize + 1, pr, idx + 1, ir);
    return root;
  }
  return dfs(0, preorder.length - 1, 0, inorder.length - 1);
}
```

## 53. 路径总和 III

- **标签**：树 / 前缀和
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

树上也能做前缀和：记录从根到当前节点的路径和出现次数。

### 代码（JavaScript）

```js
function pathSum(root, targetSum) {
  const map = new Map([[0, 1]]);
  let ans = 0;
  function dfs(node, sum) {
    if (!node) return;
    sum += node.val;
    ans += map.get(sum - targetSum) || 0;
    map.set(sum, (map.get(sum) || 0) + 1);
    dfs(node.left, sum);
    dfs(node.right, sum);
    map.set(sum, map.get(sum) - 1);
  }
  dfs(root, 0);
  return ans;
}
```

## 54. 二叉树中的最大路径和

- **标签**：树 / DFS
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

递归返回“以当前节点为起点向下延伸的最大贡献”，同时更新全局答案。

### 代码（JavaScript）

```js
function maxPathSum(root) {
  let ans = -Infinity;
  function dfs(node) {
    if (!node) return 0;
    const left = Math.max(0, dfs(node.left));
    const right = Math.max(0, dfs(node.right));
    ans = Math.max(ans, node.val + left + right);
    return node.val + Math.max(left, right);
  }
  dfs(root);
  return ans;
}
```

## 55. 二叉树的最近公共祖先

- **标签**：树 / DFS
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

如果左右子树分别找到不同目标，当前节点就是最近公共祖先。

### 代码（JavaScript）

```js
function lowestCommonAncestor(root, p, q) {
  if (!root || root === p || root === q) return root;
  const left = lowestCommonAncestor(root.left, p, q);
  const right = lowestCommonAncestor(root.right, p, q);
  if (left && right) return root;
  return left || right;
}
```

## 56. 课程表

- **标签**：图 / 拓扑排序
- **复杂度**：时间 O(V+E)，空间 O(V+E)

### 思路

统计入度，入度为 0 的节点先入队，最后看能否处理完所有课程。

### 代码（JavaScript）

```js
function canFinish(numCourses, prerequisites) {
  const graph = Array.from({ length: numCourses }, () => []);
  const indegree = new Array(numCourses).fill(0);
  for (const [a, b] of prerequisites) {
    graph[b].push(a);
    indegree[a]++;
  }
  const queue = [];
  for (let i = 0; i < numCourses; i++) if (indegree[i] === 0) queue.push(i);
  let count = 0;
  while (queue.length) {
    const cur = queue.shift();
    count++;
    for (const next of graph[cur]) {
      if (--indegree[next] === 0) queue.push(next);
    }
  }
  return count === numCourses;
}
```

## 57. 课程表 II

- **标签**：图 / 拓扑排序
- **复杂度**：时间 O(V+E)，空间 O(V+E)

### 思路

在拓扑排序过程中把节点加入结果数组，最后判断长度是否足够。

### 代码（JavaScript）

```js
function findOrder(numCourses, prerequisites) {
  const graph = Array.from({ length: numCourses }, () => []);
  const indegree = new Array(numCourses).fill(0);
  for (const [a, b] of prerequisites) {
    graph[b].push(a);
    indegree[a]++;
  }
  const queue = [];
  for (let i = 0; i < numCourses; i++) if (indegree[i] === 0) queue.push(i);
  const res = [];
  while (queue.length) {
    const cur = queue.shift();
    res.push(cur);
    for (const next of graph[cur]) {
      if (--indegree[next] === 0) queue.push(next);
    }
  }
  return res.length === numCourses ? res : [];
}
```

## 58. 克隆图

- **标签**：图 / DFS
- **复杂度**：时间 O(V+E)，空间 O(V)

### 思路

每个原节点只克隆一次，用 Map 建立“旧节点 -> 新节点”映射。

### 代码（JavaScript）

```js
function cloneGraph(node) {
  if (!node) return null;
  const map = new Map();
  function dfs(cur) {
    if (map.has(cur)) return map.get(cur);
    const copy = new Node(cur.val);
    map.set(cur, copy);
    for (const next of cur.neighbors) {
      copy.neighbors.push(dfs(next));
    }
    return copy;
  }
  return dfs(node);
}
```
