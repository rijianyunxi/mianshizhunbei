# 05 - 链表

链表题最怕的不是不会，而是改指针时把链表改断。做题时先画图，再决定每一步谁先存、谁后改。

## 这一类题怎么识别

- 题目主体是单链表或双链表
- 涉及反转、合并、删除、分组处理
- 容易因为头节点变化需要用哑节点

## 这类题的通用思路

常用套路就三种：哑节点、快慢指针、局部反转。能套模板时别硬写。

## 35. 反转链表

- **标签**：链表
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

用 `prev / cur / next` 三指针逐步反转。

### 代码（JavaScript）

```js
function reverseList(head) {
  let prev = null, cur = head;
  while (cur) {
    const next = cur.next;
    cur.next = prev;
    prev = cur;
    cur = next;
  }
  return prev;
}
```

## 36. 环形链表

- **标签**：快慢指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

快指针每次两步，慢指针每次一步，能相遇则存在环。

### 代码（JavaScript）

```js
function hasCycle(head) {
  let slow = head, fast = head;
  while (fast && fast.next) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow === fast) return true;
  }
  return false;
}
```

## 37. 环形链表 II

- **标签**：快慢指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

相遇后一个指针回到头部，再同步前进，相遇点就是入口。

### 代码（JavaScript）

```js
function detectCycle(head) {
  let slow = head, fast = head;
  while (fast && fast.next) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow === fast) {
      let p = head;
      while (p !== slow) {
        p = p.next;
        slow = slow.next;
      }
      return p;
    }
  }
  return null;
}
```

## 38. 相交链表

- **标签**：链表
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

两个指针分别走 A+B 和 B+A，天然对齐长度。

### 代码（JavaScript）

```js
function getIntersectionNode(headA, headB) {
  let p1 = headA, p2 = headB;
  while (p1 !== p2) {
    p1 = p1 ? p1.next : headB;
    p2 = p2 ? p2.next : headA;
  }
  return p1;
}
```

## 39. 合并两个有序链表

- **标签**：链表
- **复杂度**：时间 O(m+n)，空间 O(1)

### 思路

用哑节点把较小值依次接到结果链表后面。

### 代码（JavaScript）

```js
function mergeTwoLists(list1, list2) {
  const dummy = new ListNode(0);
  let cur = dummy;
  while (list1 && list2) {
    if (list1.val < list2.val) {
      cur.next = list1;
      list1 = list1.next;
    } else {
      cur.next = list2;
      list2 = list2.next;
    }
    cur = cur.next;
  }
  cur.next = list1 || list2;
  return dummy.next;
}
```

## 40. 两两交换链表中的节点

- **标签**：链表
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

每次交换相邻两个节点，哑节点让头节点变化也好处理。

### 代码（JavaScript）

```js
function swapPairs(head) {
  const dummy = new ListNode(0, head);
  let cur = dummy;
  while (cur.next && cur.next.next) {
    const first = cur.next;
    const second = first.next;
    first.next = second.next;
    second.next = first;
    cur.next = second;
    cur = first;
  }
  return dummy.next;
}
```

## 41. K 个一组翻转链表

- **标签**：链表
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

每次先检查是否够 k 个，够的话局部反转再接回主链。

### 代码（JavaScript）

```js
function reverseKGroup(head, k) {
  const dummy = new ListNode(0, head);
  let pre = dummy;
  while (true) {
    let tail = pre;
    for (let i = 0; i < k && tail; i++) tail = tail.next;
    if (!tail) break;
    const next = tail.next;
    let prev = next, cur = pre.next;
    while (cur !== next) {
      const tmp = cur.next;
      cur.next = prev;
      prev = cur;
      cur = tmp;
    }
    const start = pre.next;
    pre.next = tail;
    pre = start;
  }
  return dummy.next;
}
```

## 42. 排序链表

- **标签**：链表 / 归并排序
- **复杂度**：时间 O(n log n)，空间 O(log n)

### 思路

链表不适合随机访问，排序优先用归并。

### 代码（JavaScript）

```js
function sortList(head) {
  if (!head || !head.next) return head;
  let slow = head, fast = head.next;
  while (fast && fast.next) {
    slow = slow.next;
    fast = fast.next.next;
  }
  const rightHead = slow.next;
  slow.next = null;
  return merge(sortList(head), sortList(rightHead));
}
function merge(a, b) {
  const dummy = new ListNode(0);
  let cur = dummy;
  while (a && b) {
    if (a.val < b.val) {
      cur.next = a;
      a = a.next;
    } else {
      cur.next = b;
      b = b.next;
    }
    cur = cur.next;
  }
  cur.next = a || b;
  return dummy.next;
}
```
