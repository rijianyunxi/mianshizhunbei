# 02 - 双指针

双指针最核心的价值是：**利用有序性、相对位置关系或者原地交换，把原本双重循环的问题压到线性复杂度。**

## 这一类题怎么识别

- 数组已排序或可以先排序
- 题目要求原地处理数组
- 需要从两端向中间逼近最优解

## 这类题的通用思路

先明确两个指针分别代表什么：左右边界、快慢位置，还是“已处理区”和“未处理区”。

## 11. 盛最多水的容器

- **标签**：双指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

面积由较短板决定，所以每次移动较短的一边才可能变大。

### 代码（JavaScript）

```js
function maxArea(height) {
  let l = 0, r = height.length - 1;
  let ans = 0;
  while (l < r) {
    ans = Math.max(ans, Math.min(height[l], height[r]) * (r - l));
    if (height[l] < height[r]) l++;
    else r--;
  }
  return ans;
}
```

## 12. 三数之和

- **标签**：排序 / 双指针
- **复杂度**：时间 O(n^2)，空间 O(1) 或 O(log n)

### 思路

固定一个数后，在后面的有序区间里用双指针找另外两个数。

### 代码（JavaScript）

```js
function threeSum(nums) {
  nums.sort((a, b) => a - b);
  const res = [];
  for (let i = 0; i < nums.length - 2; i++) {
    if (i > 0 && nums[i] === nums[i - 1]) continue;
    let l = i + 1, r = nums.length - 1;
    while (l < r) {
      const sum = nums[i] + nums[l] + nums[r];
      if (sum === 0) {
        res.push([nums[i], nums[l], nums[r]]);
        while (l < r && nums[l] === nums[l + 1]) l++;
        while (l < r && nums[r] === nums[r - 1]) r--;
        l++; r--;
      } else if (sum < 0) l++;
      else r--;
    }
  }
  return res;
}
```

## 13. 移动零

- **标签**：快慢指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

慢指针表示下一个非零元素应放的位置，快指针负责扫描。

### 代码（JavaScript）

```js
function moveZeroes(nums) {
  let slow = 0;
  for (let fast = 0; fast < nums.length; fast++) {
    if (nums[fast] !== 0) {
      [nums[slow], nums[fast]] = [nums[fast], nums[slow]];
      slow++;
    }
  }
  return nums;
}
```

## 14. 接雨水

- **标签**：双指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

维护左右两边最高柱子，较低那一侧可以立即结算积水量。

### 代码（JavaScript）

```js
function trap(height) {
  let l = 0, r = height.length - 1;
  let leftMax = 0, rightMax = 0, ans = 0;
  while (l < r) {
    leftMax = Math.max(leftMax, height[l]);
    rightMax = Math.max(rightMax, height[r]);
    if (leftMax < rightMax) {
      ans += leftMax - height[l];
      l++;
    } else {
      ans += rightMax - height[r];
      r--;
    }
  }
  return ans;
}
```

## 15. 颜色分类

- **标签**：三指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

维护 `[0, zero)` 全是 0，`(two, end]` 全是 2，中间区域继续扫描。

### 代码（JavaScript）

```js
function sortColors(nums) {
  let zero = 0, i = 0, two = nums.length - 1;
  while (i <= two) {
    if (nums[i] === 0) {
      [nums[i], nums[zero]] = [nums[zero], nums[i]];
      i++;
      zero++;
    } else if (nums[i] === 2) {
      [nums[i], nums[two]] = [nums[two], nums[i]];
      two--;
    } else {
      i++;
    }
  }
  return nums;
}
```

## 16. 回文链表

- **标签**：链表 / 双指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

找中点、反转后半段，再从两头向中间比。

### 代码（JavaScript）

```js
function isPalindrome(head) {
  let slow = head, fast = head;
  while (fast && fast.next) {
    slow = slow.next;
    fast = fast.next.next;
  }
  let prev = null;
  while (slow) {
    const next = slow.next;
    slow.next = prev;
    prev = slow;
    slow = next;
  }
  while (prev) {
    if (head.val !== prev.val) return false;
    head = head.next;
    prev = prev.next;
  }
  return true;
}
```

## 17. 删除链表的倒数第 N 个结点

- **标签**：链表 / 双指针
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

快指针先走 n 步，再和慢指针一起走，慢指针最后停在待删节点前面。

### 代码（JavaScript）

```js
function removeNthFromEnd(head, n) {
  const dummy = new ListNode(0, head);
  let fast = dummy, slow = dummy;
  for (let i = 0; i < n; i++) fast = fast.next;
  while (fast.next) {
    fast = fast.next;
    slow = slow.next;
  }
  slow.next = slow.next.next;
  return dummy.next;
}
```

## 18. 有序数组的平方

- **标签**：双指针
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

绝对值最大的元素一定在两端，从结果数组尾部开始填。

### 代码（JavaScript）

```js
function sortedSquares(nums) {
  let l = 0, r = nums.length - 1;
  const res = new Array(nums.length);
  let idx = nums.length - 1;
  while (l <= r) {
    const left = nums[l] * nums[l];
    const right = nums[r] * nums[r];
    if (left > right) {
      res[idx--] = left;
      l++;
    } else {
      res[idx--] = right;
      r--;
    }
  }
  return res;
}
```
