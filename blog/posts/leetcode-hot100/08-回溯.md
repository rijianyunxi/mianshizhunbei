# 08 - 回溯

回溯题要有“树形决策”的感觉。每往下一层，就是在做一次选择；回来的时候，一定要撤销选择。

## 这一类题怎么识别

- 题目要求列举所有方案
- 有组合、排列、切割、棋盘搜索
- 通常会出现“路径 path、起点 start、已使用 used”

## 这类题的通用思路

通用模板：定义路径 -> 枚举选择 -> 做选择 -> 递归 -> 撤销选择。剪枝决定效率上限。

## 59. 全排列

- **标签**：回溯
- **复杂度**：时间 O(n! * n)，空间 O(n)

### 思路

用 `used` 数组标记哪些元素已进入路径。

### 代码（JavaScript）

```js
function permute(nums) {
  const res = [];
  const path = [];
  const used = new Array(nums.length).fill(false);
  function dfs() {
    if (path.length === nums.length) {
      res.push([...path]);
      return;
    }
    for (let i = 0; i < nums.length; i++) {
      if (used[i]) continue;
      used[i] = true;
      path.push(nums[i]);
      dfs();
      path.pop();
      used[i] = false;
    }
  }
  dfs();
  return res;
}
```

## 60. 子集

- **标签**：回溯
- **复杂度**：时间 O(n * 2^n)，空间 O(n)

### 思路

每个元素都有选或不选两种状态，因此递归时先收集当前路径。

### 代码（JavaScript）

```js
function subsets(nums) {
  const res = [];
  const path = [];
  function dfs(start) {
    res.push([...path]);
    for (let i = start; i < nums.length; i++) {
      path.push(nums[i]);
      dfs(i + 1);
      path.pop();
    }
  }
  dfs(0);
  return res;
}
```

## 61. 电话号码的字母组合

- **标签**：回溯 / 字符串
- **复杂度**：时间 O(4^n)，空间 O(n)

### 思路

每一位数字对应若干候选字符，逐层构造答案。

### 代码（JavaScript）

```js
function letterCombinations(digits) {
  if (!digits.length) return [];
  const map = {
    2: 'abc', 3: 'def', 4: 'ghi', 5: 'jkl',
    6: 'mno', 7: 'pqrs', 8: 'tuv', 9: 'wxyz'
  };
  const res = [];
  function dfs(index, path) {
    if (index === digits.length) {
      res.push(path);
      return;
    }
    for (const ch of map[digits[index]]) dfs(index + 1, path + ch);
  }
  dfs(0, '');
  return res;
}
```

## 62. 组合总和

- **标签**：回溯
- **复杂度**：时间 取决于搜索树，空间 O(target/k)

### 思路

元素可重复选，所以进入下一层时仍从当前下标开始。

### 代码（JavaScript）

```js
function combinationSum(candidates, target) {
  const res = [];
  const path = [];
  function dfs(start, remain) {
    if (remain === 0) {
      res.push([...path]);
      return;
    }
    if (remain < 0) return;
    for (let i = start; i < candidates.length; i++) {
      path.push(candidates[i]);
      dfs(i, remain - candidates[i]);
      path.pop();
    }
  }
  dfs(0, target);
  return res;
}
```

## 63. 括号生成

- **标签**：回溯
- **复杂度**：时间 为卡特兰数级别，空间 O(n)

### 思路

左括号没用完就能放，右括号数量永远不能超过左括号。

### 代码（JavaScript）

```js
function generateParenthesis(n) {
  const res = [];
  function dfs(path, left, right) {
    if (path.length === 2 * n) {
      res.push(path);
      return;
    }
    if (left < n) dfs(path + '(', left + 1, right);
    if (right < left) dfs(path + ')', left, right + 1);
  }
  dfs('', 0, 0);
  return res;
}
```

## 64. 分割回文串

- **标签**：回溯 / 字符串
- **复杂度**：时间 指数级，空间 O(n)

### 思路

从当前位置枚举所有前缀，只要是回文串就能继续往下切。

### 代码（JavaScript）

```js
function partition(s) {
  const res = [];
  const path = [];
  function isPal(str, l, r) {
    while (l < r) if (str[l++] !== str[r--]) return false;
    return true;
  }
  function dfs(start) {
    if (start === s.length) {
      res.push([...path]);
      return;
    }
    for (let end = start; end < s.length; end++) {
      if (!isPal(s, start, end)) continue;
      path.push(s.slice(start, end + 1));
      dfs(end + 1);
      path.pop();
    }
  }
  dfs(0);
  return res;
}
```

## 65. N 皇后

- **标签**：回溯 / 棋盘
- **复杂度**：时间 指数级，空间 O(n)

### 思路

逐行放皇后，列、主对角线、副对角线都不能冲突。

### 代码（JavaScript）

```js
function solveNQueens(n) {
  const res = [];
  const board = Array.from({ length: n }, () => Array(n).fill('.'));
  const cols = new Set(), diag1 = new Set(), diag2 = new Set();
  function dfs(row) {
    if (row === n) {
      res.push(board.map(row => row.join('')));
      return;
    }
    for (let col = 0; col < n; col++) {
      if (cols.has(col) || diag1.has(row - col) || diag2.has(row + col)) continue;
      cols.add(col); diag1.add(row - col); diag2.add(row + col);
      board[row][col] = 'Q';
      dfs(row + 1);
      board[row][col] = '.';
      cols.delete(col); diag1.delete(row - col); diag2.delete(row + col);
    }
  }
  dfs(0);
  return res;
}
```

## 66. 复原 IP 地址

- **标签**：回溯
- **复杂度**：时间 常数级搜索，空间 O(1)

### 思路

总共切成 4 段，每段长度 1~3，且要满足数值合法、无前导 0。

### 代码（JavaScript）

```js
function restoreIpAddresses(s) {
  const res = [];
  const path = [];
  function dfs(start) {
    if (path.length === 4) {
      if (start === s.length) res.push(path.join('.'));
      return;
    }
    for (let len = 1; len <= 3; len++) {
      if (start + len > s.length) break;
      const part = s.slice(start, start + len);
      if ((part.length > 1 && part[0] === '0') || Number(part) > 255) continue;
      path.push(part);
      dfs(start + len);
      path.pop();
    }
  }
  dfs(0);
  return res;
}
```
