# 04 - 栈与单调栈

栈题最常见的两种信号：**需要回退**，或者 **需要知道“上一个/下一个更大更小元素”**。后者往往就是单调栈。

## 这一类题怎么识别

- 括号匹配、表达式求值、字符串展开
- 要找“下一个更大元素”或“最近更小元素”
- 当前元素会让之前一批元素的答案确定

## 这类题的通用思路

普通栈解决匹配和回退；单调栈解决“谁在谁左边/右边更大更小”。

## 27. 有效的括号

- **标签**：栈
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

左括号入栈，右括号出现时必须和栈顶匹配。

### 代码（JavaScript）

```js
function isValid(s) {
  const stack = [];
  const map = { ')': '(', ']': '[', '}': '{' };
  for (const ch of s) {
    if (ch === '(' || ch === '[' || ch === '{') stack.push(ch);
    else if (stack.pop() !== map[ch]) return false;
  }
  return stack.length === 0;
}
```

## 28. 最小栈

- **标签**：设计 / 栈
- **复杂度**：各操作 O(1)，空间 O(n)

### 思路

一个栈存值，一个同步栈存到当前位置为止的最小值。

### 代码（JavaScript）

```js
class MinStack {
  constructor() {
    this.stack = [];
    this.minStack = [];
  }
  push(val) {
    this.stack.push(val);
    const min = this.minStack.length
      ? Math.min(val, this.minStack[this.minStack.length - 1])
      : val;
    this.minStack.push(min);
  }
  pop() { this.minStack.pop(); return this.stack.pop(); }
  top() { return this.stack[this.stack.length - 1]; }
  getMin() { return this.minStack[this.minStack.length - 1]; }
}
```

## 29. 字符串解码

- **标签**：栈 / 字符串
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

数字和当前字符串段分别入栈，遇到 `]` 时回退拼接。

### 代码（JavaScript）

```js
function decodeString(s) {
  const numStack = [];
  const strStack = [];
  let num = 0, cur = '';
  for (const ch of s) {
    if (ch >= '0' && ch <= '9') num = num * 10 + Number(ch);
    else if (ch === '[') {
      numStack.push(num);
      strStack.push(cur);
      num = 0;
      cur = '';
    } else if (ch === ']') {
      const repeat = numStack.pop();
      cur = strStack.pop() + cur.repeat(repeat);
    } else cur += ch;
  }
  return cur;
}
```

## 30. 每日温度

- **标签**：单调栈
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

栈里存“还没找到更高温度”的下标，遇到更高温度就结算。

### 代码（JavaScript）

```js
function dailyTemperatures(temperatures) {
  const stack = [];
  const res = new Array(temperatures.length).fill(0);
  for (let i = 0; i < temperatures.length; i++) {
    while (stack.length && temperatures[i] > temperatures[stack[stack.length - 1]]) {
      const idx = stack.pop();
      res[idx] = i - idx;
    }
    stack.push(i);
  }
  return res;
}
```

## 31. 柱状图中最大的矩形

- **标签**：单调栈
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

维护递增高度栈，当前高度一旦变小，就以弹出的柱子为高结算面积。

### 代码（JavaScript）

```js
function largestRectangleArea(heights) {
  const arr = [0, ...heights, 0];
  const stack = [];
  let ans = 0;
  for (let i = 0; i < arr.length; i++) {
    while (stack.length && arr[i] < arr[stack[stack.length - 1]]) {
      const h = arr[stack.pop()];
      const w = i - stack[stack.length - 1] - 1;
      ans = Math.max(ans, h * w);
    }
    stack.push(i);
  }
  return ans;
}
```

## 32. 接雨水（单调栈解法）

- **标签**：单调栈
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

当出现更高柱子时，可以把中间凹槽这一层的积水量结算出来。

### 代码（JavaScript）

```js
function trap(height) {
  const stack = [];
  let ans = 0;
  for (let i = 0; i < height.length; i++) {
    while (stack.length && height[i] > height[stack[stack.length - 1]]) {
      const mid = stack.pop();
      if (!stack.length) break;
      const left = stack[stack.length - 1];
      const w = i - left - 1;
      const h = Math.min(height[left], height[i]) - height[mid];
      ans += w * h;
    }
    stack.push(i);
  }
  return ans;
}
```

## 33. 逆波兰表达式求值

- **标签**：栈
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

遇到数字入栈，遇到运算符就弹出两个数计算后再压回去。

### 代码（JavaScript）

```js
function evalRPN(tokens) {
  const stack = [];
  for (const token of tokens) {
    if (!'+-*/'.includes(token) || token.length > 1) {
      stack.push(Number(token));
    } else {
      const b = stack.pop();
      const a = stack.pop();
      if (token === '+') stack.push(a + b);
      if (token === '-') stack.push(a - b);
      if (token === '*') stack.push(a * b);
      if (token === '/') stack.push(a / b > 0 ? Math.floor(a / b) : Math.ceil(a / b));
    }
  }
  return stack.pop();
}
```

## 34. 基本计算器 II

- **标签**：栈 / 字符串
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

加减先入栈，乘除立即和栈顶计算，最后求和。

### 代码（JavaScript）

```js
function calculate(s) {
  const stack = [];
  let num = 0, sign = '+';
  for (let i = 0; i <= s.length; i++) {
    const ch = s[i] || '#';
    if (ch >= '0' && ch <= '9') num = num * 10 + Number(ch);
    if ((ch < '0' || ch > '9') && ch !== ' ') {
      if (sign === '+') stack.push(num);
      if (sign === '-') stack.push(-num);
      if (sign === '*') stack.push(stack.pop() * num);
      if (sign === '/') {
        const prev = stack.pop();
        stack.push(prev / num > 0 ? Math.floor(prev / num) : Math.ceil(prev / num));
      }
      sign = ch;
      num = 0;
    }
  }
  return stack.reduce((a, b) => a + b, 0);
}
```
