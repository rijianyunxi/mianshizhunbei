# 01 - 哈希与数组（10 题）

这一篇先把最基础、也最常考的哈希与数组题吃透。核心就是：**要么用哈希降低查找成本，要么利用下标、前缀和、原地交换挖掘数组结构信息。**

## 1. 两数之和

- **标签**：哈希表 / 数组

### 思路

遍历数组时把“值 -> 下标”存进 Map。当前数字只要能在 Map 里找到 `target - nums[i]`，就说明答案已经出现。

### 代码（JavaScript）

```js
function twoSum(nums, target) {
  const map = new Map();
  for (let i = 0; i < nums.length; i++) {
    const need = target - nums[i];
    if (map.has(need)) return [map.get(need), i];
    map.set(nums[i], i);
  }
  return [];
}
```

## 2. 字母异位词分组

- **标签**：哈希表 / 字符串

### 思路

把每个字符串排序后的结果作为 key，相同 key 的字符串放进同一组。

### 代码（JavaScript）

```js
function groupAnagrams(strs) {
  const map = new Map();
  for (const str of strs) {
    const key = str.split('').sort().join('');
    if (!map.has(key)) map.set(key, []);
    map.get(key).push(str);
  }
  return [...map.values()];
}
```

## 3. 最长连续序列

- **标签**：哈希表

### 思路

先把所有数放进 Set。只有当 `num - 1` 不存在时，才从这个数开始向后扩展连续段。

### 代码（JavaScript）

```js
function longestConsecutive(nums) {
  const set = new Set(nums);
  let ans = 0;
  for (const num of set) {
    if (set.has(num - 1)) continue;
    let cur = num;
    let len = 1;
    while (set.has(cur + 1)) {
      cur++;
      len++;
    }
    ans = Math.max(ans, len);
  }
  return ans;
}
```

## 4. 多数元素

- **标签**：数组 / 投票算法

### 思路

用 Boyer-Moore 投票法维护候选人和票数。不同就抵消，相同就加票。

### 代码（JavaScript）

```js
function majorityElement(nums) {
  let candidate = null;
  let count = 0;
  for (const num of nums) {
    if (count === 0) candidate = num;
    count += num === candidate ? 1 : -1;
  }
  return candidate;
}
```

## 5. 除自身以外数组的乘积

- **标签**：数组 / 前后缀

### 思路

第一遍记录每个位置左边所有数的乘积，第二遍从右往左补上右边乘积。

### 代码（JavaScript）

```js
function productExceptSelf(nums) {
  const res = new Array(nums.length).fill(1);
  let prefix = 1;
  for (let i = 0; i < nums.length; i++) {
    res[i] = prefix;
    prefix *= nums[i];
  }
  let suffix = 1;
  for (let i = nums.length - 1; i >= 0; i--) {
    res[i] *= suffix;
    suffix *= nums[i];
  }
  return res;
}
```

## 6. 缺失的第一个正数

- **标签**：数组 / 原地哈希

### 思路

理想状态下值 `x` 应该放在下标 `x - 1`。不断交换到正确位置，最后找第一个不匹配的位置。

### 代码（JavaScript）

```js
function firstMissingPositive(nums) {
  const n = nums.length;
  for (let i = 0; i < n; i++) {
    while (
      nums[i] >= 1 && nums[i] <= n &&
      nums[nums[i] - 1] !== nums[i]
    ) {
      const idx = nums[i] - 1;
      [nums[i], nums[idx]] = [nums[idx], nums[i]];
    }
  }
  for (let i = 0; i < n; i++) {
    if (nums[i] !== i + 1) return i + 1;
  }
  return n + 1;
}
```

## 7. 找到所有数组中消失的数字

- **标签**：数组 / 原地标记

### 思路

把出现过的数字对应下标位置标记成负数，最后仍为正数的位置就是没出现过的数。

### 代码（JavaScript）

```js
function findDisappearedNumbers(nums) {
  for (let i = 0; i < nums.length; i++) {
    const idx = Math.abs(nums[i]) - 1;
    if (nums[idx] > 0) nums[idx] = -nums[idx];
  }
  const res = [];
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] > 0) res.push(i + 1);
  }
  return res;
}
```

## 8. 子数组和等于 K

- **标签**：前缀和 / 哈希表

### 思路

遍历时维护前缀和 `sum`。如果之前出现过 `sum - k`，就说明从那之后到当前这段和为 `k`。

### 代码（JavaScript）

```js
function subarraySum(nums, k) {
  const map = new Map();
  map.set(0, 1);
  let sum = 0;
  let ans = 0;
  for (const num of nums) {
    sum += num;
    if (map.has(sum - k)) ans += map.get(sum - k);
    map.set(sum, (map.get(sum) || 0) + 1);
  }
  return ans;
}
```

## 9. 和为 K 的子数组个数

- **标签**：前缀和 / 哈希表

### 思路

本质和上一题一样，关键点是前缀和频次统计。

### 代码（JavaScript）

```js
function countSubarraysEqualK(nums, k) {
  const map = new Map([[0, 1]]);
  let sum = 0, ans = 0;
  for (const num of nums) {
    sum += num;
    ans += map.get(sum - k) || 0;
    map.set(sum, (map.get(sum) || 0) + 1);
  }
  return ans;
}
```

## 10. 前 K 个高频元素

- **标签**：哈希表 / 堆/排序

### 思路

先统计频次，再按频次排序或维护堆。JS 里面试写法用排序最直接。

### 代码（JavaScript）

```js
function topKFrequent(nums, k) {
  const map = new Map();
  for (const num of nums) {
    map.set(num, (map.get(num) || 0) + 1);
  }
  return [...map.entries()]
    .sort((a, b) => b[1] - a[1])
    .slice(0, k)
    .map(item => item[0]);
}
```
