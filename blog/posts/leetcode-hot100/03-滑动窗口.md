# 03 - 滑动窗口

滑动窗口本质是在维护一个动态区间。只要题目里出现“子串 / 子数组 / 连续区间 / 最长最短满足条件”这些词，就要警惕滑窗。

## 这一类题怎么识别

- 要求处理连续区间
- 需要找最长 / 最短 / 个数
- 窗口状态可以在左右指针移动时增量维护

## 这类题的通用思路

一套固定流程：右指针扩张 -> 更新窗口状态 -> 当窗口不合法时左指针收缩 -> 更新答案。

## 19. 无重复字符的最长子串

- **标签**：滑动窗口 / 哈希表
- **复杂度**：时间 O(n)，空间 O(字符集大小)

### 思路

窗口内不允许字符重复，重复时不断收缩左边界直到恢复合法。

### 代码（JavaScript）

```js
function lengthOfLongestSubstring(s) {
  const map = new Map();
  let left = 0, ans = 0;
  for (let right = 0; right < s.length; right++) {
    const ch = s[right];
    map.set(ch, (map.get(ch) || 0) + 1);
    while (map.get(ch) > 1) {
      const leftChar = s[left++];
      map.set(leftChar, map.get(leftChar) - 1);
    }
    ans = Math.max(ans, right - left + 1);
  }
  return ans;
}
```

## 20. 找到字符串中所有字母异位词

- **标签**：滑动窗口
- **复杂度**：时间 O(n * 26)，空间 O(26)

### 思路

维护固定长度窗口和字符频次数组，窗口长度等于 p 时比较状态。

### 代码（JavaScript）

```js
function findAnagrams(s, p) {
  const need = new Array(26).fill(0);
  const win = new Array(26).fill(0);
  for (const ch of p) need[ch.charCodeAt(0) - 97]++;
  const res = [];
  let left = 0;
  for (let right = 0; right < s.length; right++) {
    win[s.charCodeAt(right) - 97]++;
    if (right - left + 1 > p.length) {
      win[s.charCodeAt(left) - 97]--;
      left++;
    }
    if (right - left + 1 === p.length && need.join(',') === win.join(',')) {
      res.push(left);
    }
  }
  return res;
}
```

## 21. 最小覆盖子串

- **标签**：滑动窗口 / 哈希表
- **复杂度**：时间 O(n)，空间 O(字符集大小)

### 思路

先扩张到满足覆盖条件，再尽量缩小窗口更新最短答案。

### 代码（JavaScript）

```js
function minWindow(s, t) {
  const need = new Map();
  for (const ch of t) need.set(ch, (need.get(ch) || 0) + 1);
  let needType = need.size;
  let left = 0, start = 0, minLen = Infinity;
  for (let right = 0; right < s.length; right++) {
    const ch = s[right];
    if (need.has(ch)) {
      need.set(ch, need.get(ch) - 1);
      if (need.get(ch) === 0) needType--;
    }
    while (needType === 0) {
      if (right - left + 1 < minLen) {
        minLen = right - left + 1;
        start = left;
      }
      const leftChar = s[left++];
      if (need.has(leftChar)) {
        if (need.get(leftChar) === 0) needType++;
        need.set(leftChar, need.get(leftChar) + 1);
      }
    }
  }
  return minLen === Infinity ? '' : s.slice(start, start + minLen);
}
```

## 22. 滑动窗口最大值

- **标签**：滑动窗口 / 单调队列
- **复杂度**：时间 O(n)，空间 O(k)

### 思路

队列中保存下标并保持值递减，队头始终是窗口最大值。

### 代码（JavaScript）

```js
function maxSlidingWindow(nums, k) {
  const deque = [];
  const res = [];
  for (let i = 0; i < nums.length; i++) {
    while (deque.length && nums[deque[deque.length - 1]] <= nums[i]) deque.pop();
    deque.push(i);
    if (deque[0] <= i - k) deque.shift();
    if (i >= k - 1) res.push(nums[deque[0]]);
  }
  return res;
}
```

## 23. 长度最小的子数组

- **标签**：滑动窗口
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

窗口和一旦达到目标，就尽量缩小左边界。

### 代码（JavaScript）

```js
function minSubArrayLen(target, nums) {
  let left = 0, sum = 0, ans = Infinity;
  for (let right = 0; right < nums.length; right++) {
    sum += nums[right];
    while (sum >= target) {
      ans = Math.min(ans, right - left + 1);
      sum -= nums[left++];
    }
  }
  return ans === Infinity ? 0 : ans;
}
```

## 24. 乘积小于 K 的子数组

- **标签**：滑动窗口
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

维护窗口乘积，若乘积超标就左移。当前窗口长度就是新增的合法子数组数。

### 代码（JavaScript）

```js
function numSubarrayProductLessThanK(nums, k) {
  if (k <= 1) return 0;
  let left = 0, product = 1, ans = 0;
  for (let right = 0; right < nums.length; right++) {
    product *= nums[right];
    while (product >= k) product /= nums[left++];
    ans += right - left + 1;
  }
  return ans;
}
```

## 25. 字符串的排列

- **标签**：滑动窗口
- **复杂度**：时间 O(n * 26)，空间 O(26)

### 思路

维护长度为 `s1.length` 的窗口，判断它是否和 s1 具有相同字符频次。

### 代码（JavaScript）

```js
function checkInclusion(s1, s2) {
  const need = new Array(26).fill(0);
  const win = new Array(26).fill(0);
  for (const ch of s1) need[ch.charCodeAt(0) - 97]++;
  let left = 0;
  for (let right = 0; right < s2.length; right++) {
    win[s2.charCodeAt(right) - 97]++;
    if (right - left + 1 > s1.length) {
      win[s2.charCodeAt(left) - 97]--;
      left++;
    }
    if (right - left + 1 === s1.length && need.join(',') === win.join(',')) return true;
  }
  return false;
}
```

## 26. 最大连续 1 的个数 III

- **标签**：滑动窗口
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

窗口里最多允许出现 k 个 0，超过就缩左边界。

### 代码（JavaScript）

```js
function longestOnes(nums, k) {
  let left = 0, zero = 0, ans = 0;
  for (let right = 0; right < nums.length; right++) {
    if (nums[right] === 0) zero++;
    while (zero > k) {
      if (nums[left++] === 0) zero--;
    }
    ans = Math.max(ans, right - left + 1);
  }
  return ans;
}
```
