# 03 - 栈与队列（10 题）

这一组题最重要的是识别：**什么时候需要“后进先出”的回退能力，什么时候需要“保持单调”的结构信息。**

## 21. 有效的括号

- **标签**：栈

### 思路

遇到左括号入栈，遇到右括号时检查栈顶是否为对应左括号。

### 代码（JavaScript）

```js
function isValid(s) {
  const stack = [];
  const map = { ')': '(', ']': '[', '}': '{' };
  for (const ch of s) {
    if (ch === '(' || ch === '[' || ch === '{') stack.push(ch);
    else if (stack.pop() !== map[ch]) return false;
  }
  return stack.length === 0;
}
```

## 22. 最小栈

- **标签**：设计 / 栈

### 思路

普通栈存数据，辅助栈同步存每一步的最小值。

### 代码（JavaScript）

```js
class MinStack {
  constructor() {
    this.stack = [];
    this.minStack = [];
  }
  push(val) {
    this.stack.push(val);
    const min = this.minStack.length
      ? Math.min(val, this.minStack[this.minStack.length - 1])
      : val;
    this.minStack.push(min);
  }
  pop() {
    this.minStack.pop();
    return this.stack.pop();
  }
  top() {
    return this.stack[this.stack.length - 1];
  }
  getMin() {
    return this.minStack[this.minStack.length - 1];
  }
}
```

## 23. 字符串解码

- **标签**：栈 / 字符串

### 思路

数字和当前字符串段分别入栈，遇到 `]` 时回退并重复拼接。

### 代码（JavaScript）

```js
function decodeString(s) {
  const numStack = [];
  const strStack = [];
  let num = 0, cur = '';
  for (const ch of s) {
    if (ch >= '0' && ch <= '9') num = num * 10 + Number(ch);
    else if (ch === '[') {
      numStack.push(num);
      strStack.push(cur);
      num = 0;
      cur = '';
    } else if (ch === ']') {
      const repeat = numStack.pop();
      cur = strStack.pop() + cur.repeat(repeat);
    } else {
      cur += ch;
    }
  }
  return cur;
}
```

## 24. 每日温度

- **标签**：单调栈

### 思路

栈中保存还没找到更高温度的位置，当前温度一旦更高就能更新那些位置的答案。

### 代码（JavaScript）

```js
function dailyTemperatures(temperatures) {
  const stack = [];
  const res = new Array(temperatures.length).fill(0);
  for (let i = 0; i < temperatures.length; i++) {
    while (stack.length && temperatures[i] > temperatures[stack[stack.length - 1]]) {
      const idx = stack.pop();
      res[idx] = i - idx;
    }
    stack.push(i);
  }
  return res;
}
```

## 25. 柱状图中最大的矩形

- **标签**：单调栈

### 思路

维护递增高度栈，一旦当前高度更小，就以弹出的柱子作为高来结算最大面积。

### 代码（JavaScript）

```js
function largestRectangleArea(heights) {
  const arr = [0, ...heights, 0];
  const stack = [];
  let ans = 0;
  for (let i = 0; i < arr.length; i++) {
    while (stack.length && arr[i] < arr[stack[stack.length - 1]]) {
      const h = arr[stack.pop()];
      const w = i - stack[stack.length - 1] - 1;
      ans = Math.max(ans, h * w);
    }
    stack.push(i);
  }
  return ans;
}
```

## 26. 用栈实现队列

- **标签**：栈 / 队列

### 思路

一个栈负责入队，另一个栈负责出队，出队栈空时再整体倒过去。

### 代码（JavaScript）

```js
class MyQueue {
  constructor() {
    this.inStack = [];
    this.outStack = [];
  }
  push(x) { this.inStack.push(x); }
  move() {
    if (!this.outStack.length) {
      while (this.inStack.length) this.outStack.push(this.inStack.pop());
    }
  }
  pop() { this.move(); return this.outStack.pop(); }
  peek() { this.move(); return this.outStack[this.outStack.length - 1]; }
  empty() { return !this.inStack.length && !this.outStack.length; }
}
```

## 27. 用队列实现栈

- **标签**：队列 / 栈

### 思路

每次 push 后把前面的元素重新转到队尾，让新元素始终位于队首。

### 代码（JavaScript）

```js
class MyStack {
  constructor() { this.q = []; }
  push(x) {
    this.q.push(x);
    for (let i = 0; i < this.q.length - 1; i++) {
      this.q.push(this.q.shift());
    }
  }
  pop() { return this.q.shift(); }
  top() { return this.q[0]; }
  empty() { return this.q.length === 0; }
}
```

## 28. 逆波兰表达式求值

- **标签**：栈

### 思路

遇到数字就入栈，遇到运算符就弹出两个数计算后再压栈。

### 代码（JavaScript）

```js
function evalRPN(tokens) {
  const stack = [];
  for (const token of tokens) {
    if (!'+-*/'.includes(token) || token.length > 1) {
      stack.push(Number(token));
    } else {
      const b = stack.pop();
      const a = stack.pop();
      if (token === '+') stack.push(a + b);
      if (token === '-') stack.push(a - b);
      if (token === '*') stack.push(a * b);
      if (token === '/') stack.push(a / b > 0 ? Math.floor(a / b) : Math.ceil(a / b));
    }
  }
  return stack.pop();
}
```

## 29. 简化路径

- **标签**：栈 / 字符串

### 思路

按 `/` 分割路径，遇到普通目录入栈，遇到 `..` 出栈，最后再拼回去。

### 代码（JavaScript）

```js
function simplifyPath(path) {
  const stack = [];
  for (const part of path.split('/')) {
    if (!part || part === '.') continue;
    if (part === '..') stack.pop();
    else stack.push(part);
  }
  return '/' + stack.join('/');
}
```

## 30. 基本计算器 II

- **标签**：栈 / 字符串

### 思路

加减先入栈，乘除立即和栈顶结算，最后把栈求和。

### 代码（JavaScript）

```js
function calculate(s) {
  const stack = [];
  let num = 0, sign = '+';
  for (let i = 0; i <= s.length; i++) {
    const ch = s[i] || '#';
    if (ch >= '0' && ch <= '9') num = num * 10 + Number(ch);
    if ((ch < '0' || ch > '9') && ch !== ' ') {
      if (sign === '+') stack.push(num);
      if (sign === '-') stack.push(-num);
      if (sign === '*') stack.push(stack.pop() * num);
      if (sign === '/') {
        const prev = stack.pop();
        stack.push(prev / num > 0 ? Math.floor(prev / num) : Math.ceil(prev / num));
      }
      sign = ch;
      num = 0;
    }
  }
  return stack.reduce((a, b) => a + b, 0);
}
```
