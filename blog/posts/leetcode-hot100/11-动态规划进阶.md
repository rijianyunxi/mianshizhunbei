# 11 - 动态规划进阶

这一组是 DP 里更常被追问“为什么这样定义状态”的题。写之前先把状态图在脑子里过一遍。

## 这一类题怎么识别

- 序列比较、字符串编辑、回文区间
- 涉及 0/1 背包、区间状态、多维状态
- 面试官喜欢追问状态压缩或优化

## 这类题的通用思路

从“最后一步发生了什么”反推状态转移，很多 DP 就顺了。

## 85. 分割等和子集

- **标签**：动态规划 / 0/1 背包
- **复杂度**：时间 O(n*sum)，空间 O(sum)

### 思路

本质是看能否选出若干数使和等于总和的一半。

### 代码（JavaScript）

```js
function canPartition(nums) {
  const sum = nums.reduce((a, b) => a + b, 0);
  if (sum % 2) return false;
  const target = sum / 2;
  const dp = new Array(target + 1).fill(false);
  dp[0] = true;
  for (const num of nums) {
    for (let j = target; j >= num; j--) {
      dp[j] = dp[j] || dp[j - num];
    }
  }
  return dp[target];
}
```

## 86. 最长有效括号

- **标签**：动态规划
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

当前位置是右括号时，尝试和前一个有效段拼接。

### 代码（JavaScript）

```js
function longestValidParentheses(s) {
  const dp = new Array(s.length).fill(0);
  let ans = 0;
  for (let i = 1; i < s.length; i++) {
    if (s[i] === ')') {
      if (s[i - 1] === '(') dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2;
      else {
        const pre = i - dp[i - 1] - 1;
        if (pre >= 0 && s[pre] === '(') {
          dp[i] = dp[i - 1] + 2 + (pre >= 1 ? dp[pre - 1] : 0);
        }
      }
      ans = Math.max(ans, dp[i]);
    }
  }
  return ans;
}
```

## 87. 最长公共子序列

- **标签**：动态规划
- **复杂度**：时间 O(mn)，空间 O(mn)

### 思路

字符相等时继承左上角加一，否则取左边和上边较大值。

### 代码（JavaScript）

```js
function longestCommonSubsequence(text1, text2) {
  const m = text1.length, n = text2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i - 1] === text2[j - 1]) dp[i][j] = dp[i - 1][j - 1] + 1;
      else dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
    }
  }
  return dp[m][n];
}
```

## 88. 编辑距离

- **标签**：动态规划
- **复杂度**：时间 O(mn)，空间 O(mn)

### 思路

当前字符不同，就在插入、删除、替换三种操作中取最优。

### 代码（JavaScript）

```js
function minDistance(word1, word2) {
  const m = word1.length, n = word2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));
  for (let i = 0; i <= m; i++) dp[i][0] = i;
  for (let j = 0; j <= n; j++) dp[0][j] = j;
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (word1[i - 1] === word2[j - 1]) dp[i][j] = dp[i - 1][j - 1];
      else dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) + 1;
    }
  }
  return dp[m][n];
}
```

## 89. 最长回文子串

- **标签**：动态规划 / 字符串
- **复杂度**：时间 O(n^2)，空间 O(n^2)

### 思路

`dp[i][j]` 表示区间是否回文，按区间长度从短到长转移。

### 代码（JavaScript）

```js
function longestPalindrome(s) {
  const n = s.length;
  const dp = Array.from({ length: n }, () => new Array(n).fill(false));
  let start = 0, maxLen = 1;
  for (let i = n - 1; i >= 0; i--) {
    for (let j = i; j < n; j++) {
      if (s[i] === s[j] && (j - i <= 1 || dp[i + 1][j - 1])) {
        dp[i][j] = true;
        if (j - i + 1 > maxLen) {
          start = i;
          maxLen = j - i + 1;
        }
      }
    }
  }
  return s.slice(start, start + maxLen);
}
```

## 90. 回文子串

- **标签**：中心扩散 / 动态规划
- **复杂度**：时间 O(n^2)，空间 O(1)

### 思路

枚举每个中心向两侧扩展，统计所有回文区间。

### 代码（JavaScript）

```js
function countSubstrings(s) {
  let ans = 0;
  function expand(l, r) {
    while (l >= 0 && r < s.length && s[l] === s[r]) {
      ans++;
      l--;
      r++;
    }
  }
  for (let i = 0; i < s.length; i++) {
    expand(i, i);
    expand(i, i + 1);
  }
  return ans;
}
```

## 91. 买卖股票的最佳时机含冷冻期

- **标签**：状态 DP
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

把每天状态拆成持有、刚卖出、冷冻/休息三类再转移。

### 代码（JavaScript）

```js
function maxProfit(prices) {
  let hold = -prices[0], sold = 0, rest = 0;
  for (let i = 1; i < prices.length; i++) {
    const prevSold = sold;
    sold = hold + prices[i];
    hold = Math.max(hold, rest - prices[i]);
    rest = Math.max(rest, prevSold);
  }
  return Math.max(sold, rest);
}
```

## 92. 买卖股票的最佳时机含手续费

- **标签**：状态 DP
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

卖出时扣手续费，状态只有持有和不持有两种。

### 代码（JavaScript）

```js
function maxProfitWithFee(prices, fee) {
  let hold = -prices[0], cash = 0;
  for (let i = 1; i < prices.length; i++) {
    cash = Math.max(cash, hold + prices[i] - fee);
    hold = Math.max(hold, cash - prices[i]);
  }
  return cash;
}
```
