# 05 - 二叉树基础（10 题）

树题写不出来，通常不是代码问题，而是**递归定义不清楚**。先问自己：这个函数要解决的是“当前节点的什么问题”？

## 41. 二叉树的中序遍历

- **标签**：二叉树 / DFS

### 思路

中序就是左根右，递归和显式栈都能写。

### 代码（JavaScript）

```js
function inorderTraversal(root) {
  const res = [];
  function dfs(node) {
    if (!node) return;
    dfs(node.left);
    res.push(node.val);
    dfs(node.right);
  }
  dfs(root);
  return res;
}
```

## 42. 二叉树的最大深度

- **标签**：二叉树 / DFS

### 思路

当前节点的最大深度等于左右子树最大深度中的较大值再加 1。

### 代码（JavaScript）

```js
function maxDepth(root) {
  if (!root) return 0;
  return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
}
```

## 43. 翻转二叉树

- **标签**：二叉树

### 思路

递归交换每个节点的左右子树。

### 代码（JavaScript）

```js
function invertTree(root) {
  if (!root) return null;
  [root.left, root.right] = [root.right, root.left];
  invertTree(root.left);
  invertTree(root.right);
  return root;
}
```

## 44. 对称二叉树

- **标签**：二叉树

### 思路

比较左子树的左节点和右子树的右节点是否一一镜像。

### 代码（JavaScript）

```js
function isSymmetric(root) {
  function check(a, b) {
    if (!a && !b) return true;
    if (!a || !b || a.val !== b.val) return false;
    return check(a.left, b.right) && check(a.right, b.left);
  }
  return check(root.left, root.right);
}
```

## 45. 二叉树的直径

- **标签**：二叉树 / DFS

### 思路

每个节点处尝试用“左深度 + 右深度”更新答案，递归返回单边最大深度。

### 代码（JavaScript）

```js
function diameterOfBinaryTree(root) {
  let ans = 0;
  function depth(node) {
    if (!node) return 0;
    const left = depth(node.left);
    const right = depth(node.right);
    ans = Math.max(ans, left + right);
    return Math.max(left, right) + 1;
  }
  depth(root);
  return ans;
}
```

## 46. 二叉树的层序遍历

- **标签**：二叉树 / BFS

### 思路

用队列按层遍历，每轮先记住当前层节点数。

### 代码（JavaScript）

```js
function levelOrder(root) {
  if (!root) return [];
  const queue = [root];
  const res = [];
  while (queue.length) {
    const size = queue.length;
    const level = [];
    for (let i = 0; i < size; i++) {
      const node = queue.shift();
      level.push(node.val);
      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }
    res.push(level);
  }
  return res;
}
```

## 47. 将有序数组转换为二叉搜索树

- **标签**：BST / 分治

### 思路

每次取中点作为根节点，左右两边递归构造子树。

### 代码（JavaScript）

```js
function sortedArrayToBST(nums) {
  function build(l, r) {
    if (l > r) return null;
    const mid = Math.floor((l + r) / 2);
    const node = new TreeNode(nums[mid]);
    node.left = build(l, mid - 1);
    node.right = build(mid + 1, r);
    return node;
  }
  return build(0, nums.length - 1);
}
```

## 48. 验证二叉搜索树

- **标签**：BST

### 思路

每个节点都必须落在合法区间 `(min, max)` 内，而不是只比较父节点。

### 代码（JavaScript）

```js
function isValidBST(root) {
  function dfs(node, min, max) {
    if (!node) return true;
    if (node.val <= min || node.val >= max) return false;
    return dfs(node.left, min, node.val) && dfs(node.right, node.val, max);
  }
  return dfs(root, -Infinity, Infinity);
}
```

## 49. 二叉搜索树中第 K 小的元素

- **标签**：BST / 中序遍历

### 思路

BST 的中序遍历是递增序列，第 k 个访问到的节点就是答案。

### 代码（JavaScript）

```js
function kthSmallest(root, k) {
  const stack = [];
  let cur = root;
  while (cur || stack.length) {
    while (cur) {
      stack.push(cur);
      cur = cur.left;
    }
    cur = stack.pop();
    if (--k === 0) return cur.val;
    cur = cur.right;
  }
}
```

## 50. 从前序与中序遍历序列构造二叉树

- **标签**：二叉树 / 分治

### 思路

前序第一个元素是根节点，再根据中序位置切出左右子树。

### 代码（JavaScript）

```js
function buildTree(preorder, inorder) {
  const indexMap = new Map();
  inorder.forEach((val, i) => indexMap.set(val, i));
  function build(pl, pr, il, ir) {
    if (pl > pr) return null;
    const rootVal = preorder[pl];
    const idx = indexMap.get(rootVal);
    const leftSize = idx - il;
    const root = new TreeNode(rootVal);
    root.left = build(pl + 1, pl + leftSize, il, idx - 1);
    root.right = build(pl + leftSize + 1, pr, idx + 1, ir);
    return root;
  }
  return build(0, preorder.length - 1, 0, inorder.length - 1);
}
```
