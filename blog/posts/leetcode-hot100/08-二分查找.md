# 08 - 二分查找（10 题）

二分题最容易错在边界。写之前先固定模板：**找某个值、找左边界、找右边界，三种写法不要混。**

## 71. 搜索插入位置

- **标签**：二分查找

### 思路

标准找左边界模板，最后的 left 就是插入位置。

### 代码（JavaScript）

```js
function searchInsert(nums, target) {
  let left = 0, right = nums.length - 1;
  while (left <= right) {
    const mid = left + ((right - left) >> 1);
    if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
  }
  return left;
}
```

## 72. 搜索二维矩阵

- **标签**：二分查找

### 思路

矩阵按行展开后整体有序，可以直接在虚拟一维数组上二分。

### 代码（JavaScript）

```js
function searchMatrix(matrix, target) {
  const m = matrix.length, n = matrix[0].length;
  let left = 0, right = m * n - 1;
  while (left <= right) {
    const mid = left + ((right - left) >> 1);
    const val = matrix[Math.floor(mid / n)][mid % n];
    if (val === target) return true;
    if (val < target) left = mid + 1;
    else right = mid - 1;
  }
  return false;
}
```

## 73. 在排序数组中查找元素的第一个和最后一个位置

- **标签**：二分查找

### 思路

分别写两个二分：一个找左边界，一个找右边界。

### 代码（JavaScript）

```js
function searchRange(nums, target) {
  function lowerBound(arr, target) {
    let l = 0, r = arr.length;
    while (l < r) {
      const mid = l + ((r - l) >> 1);
      if (arr[mid] < target) l = mid + 1;
      else r = mid;
    }
    return l;
  }
  const left = lowerBound(nums, target);
  const right = lowerBound(nums, target + 1) - 1;
  if (left >= nums.length || nums[left] !== target) return [-1, -1];
  return [left, right];
}
```

## 74. 搜索旋转排序数组

- **标签**：二分查找

### 思路

每次至少有一半区间是有序的，判断 target 落在哪一半。

### 代码（JavaScript）

```js
function search(nums, target) {
  let l = 0, r = nums.length - 1;
  while (l <= r) {
    const mid = l + ((r - l) >> 1);
    if (nums[mid] === target) return mid;
    if (nums[l] <= nums[mid]) {
      if (nums[l] <= target && target < nums[mid]) r = mid - 1;
      else l = mid + 1;
    } else {
      if (nums[mid] < target && target <= nums[r]) l = mid + 1;
      else r = mid - 1;
    }
  }
  return -1;
}
```

## 75. 寻找旋转排序数组中的最小值

- **标签**：二分查找

### 思路

拿 `mid` 和 `right` 比较，判断最小值在左半还是右半。

### 代码（JavaScript）

```js
function findMin(nums) {
  let l = 0, r = nums.length - 1;
  while (l < r) {
    const mid = l + ((r - l) >> 1);
    if (nums[mid] > nums[r]) l = mid + 1;
    else r = mid;
  }
  return nums[l];
}
```

## 76. 寻找两个正序数组的中位数

- **标签**：二分查找

### 思路

在较短数组上二分切分位置，使左右两边元素个数和大小关系都满足。

### 代码（JavaScript）

```js
function findMedianSortedArrays(nums1, nums2) {
  if (nums1.length > nums2.length) return findMedianSortedArrays(nums2, nums1);
  const m = nums1.length, n = nums2.length;
  let left = 0, right = m;
  while (left <= right) {
    const i = Math.floor((left + right) / 2);
    const j = Math.floor((m + n + 1) / 2) - i;
    const maxLeftA = i === 0 ? -Infinity : nums1[i - 1];
    const minRightA = i === m ? Infinity : nums1[i];
    const maxLeftB = j === 0 ? -Infinity : nums2[j - 1];
    const minRightB = j === n ? Infinity : nums2[j];
    if (maxLeftA <= minRightB && maxLeftB <= minRightA) {
      if ((m + n) % 2 === 1) return Math.max(maxLeftA, maxLeftB);
      return (Math.max(maxLeftA, maxLeftB) + Math.min(minRightA, minRightB)) / 2;
    }
    if (maxLeftA > minRightB) right = i - 1;
    else left = i + 1;
  }
}
```

## 77. 寻找峰值

- **标签**：二分查找

### 思路

如果 `mid < mid + 1`，说明右边存在峰值；否则左边存在峰值。

### 代码（JavaScript）

```js
function findPeakElement(nums) {
  let l = 0, r = nums.length - 1;
  while (l < r) {
    const mid = l + ((r - l) >> 1);
    if (nums[mid] < nums[mid + 1]) l = mid + 1;
    else r = mid;
  }
  return l;
}
```

## 78. 寻找重复数

- **标签**：二分查找 / 快慢指针

### 思路

除了快慢指针，还能对值域二分：统计小于等于 mid 的个数是否超标。

### 代码（JavaScript）

```js
function findDuplicate(nums) {
  let l = 1, r = nums.length - 1;
  while (l < r) {
    const mid = l + ((r - l) >> 1);
    let count = 0;
    for (const num of nums) if (num <= mid) count++;
    if (count > mid) r = mid;
    else l = mid + 1;
  }
  return l;
}
```

## 79. 开平方

- **标签**：二分查找

### 思路

在 `[0, x]` 范围二分查找最大的 `mid`，使得 `mid * mid <= x`。

### 代码（JavaScript）

```js
function mySqrt(x) {
  let l = 0, r = x;
  while (l <= r) {
    const mid = l + ((r - l) >> 1);
    if (mid * mid <= x) l = mid + 1;
    else r = mid - 1;
  }
  return r;
}
```

## 80. 寻找右区间

- **标签**：排序 / 二分查找

### 思路

先把每个区间起点和原下标排序，再对每个区间的终点做二分。

### 代码（JavaScript）

```js
function findRightInterval(intervals) {
  const starts = intervals.map((it, i) => [it[0], i]).sort((a, b) => a[0] - b[0]);
  const res = new Array(intervals.length).fill(-1);
  for (let i = 0; i < intervals.length; i++) {
    let l = 0, r = starts.length - 1, ans = -1;
    while (l <= r) {
      const mid = l + ((r - l) >> 1);
      if (starts[mid][0] >= intervals[i][1]) {
        ans = starts[mid][1];
        r = mid - 1;
      } else l = mid + 1;
    }
    res[i] = ans;
  }
  return res;
}
```
