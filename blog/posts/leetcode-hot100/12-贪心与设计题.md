# 12 - 贪心与设计题

最后这一组是前端面试里很容易被拿来拉开差距的题：一部分考贪心证明，一部分考数据结构设计。

## 这一类题怎么识别

- 题目要求“最少步数 / 最大收益 / 最优安排”
- 每一步做局部最优看起来就很自然
- 需要自己设计支持 O(1) 或均摊 O(1) 的结构

## 这类题的通用思路

贪心题要能解释“为什么局部最优可以推出全局最优”；设计题要先确定操作复杂度目标。

## 93. 买卖股票的最佳时机

- **标签**：贪心
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

遍历时维护历史最低买入价，并更新最大利润。

### 代码（JavaScript）

```js
function maxProfit(prices) {
  let minPrice = Infinity, ans = 0;
  for (const price of prices) {
    minPrice = Math.min(minPrice, price);
    ans = Math.max(ans, price - minPrice);
  }
  return ans;
}
```

## 94. 跳跃游戏

- **标签**：贪心
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

维护当前最远可达下标，只要遍历过程中不掉队，就能到终点。

### 代码（JavaScript）

```js
function canJump(nums) {
  let farthest = 0;
  for (let i = 0; i < nums.length; i++) {
    if (i > farthest) return false;
    farthest = Math.max(farthest, i + nums[i]);
  }
  return true;
}
```

## 95. 跳跃游戏 II

- **标签**：贪心
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

按层推进，每到当前边界就意味着必须再跳一步。

### 代码（JavaScript）

```js
function jump(nums) {
  let end = 0, farthest = 0, steps = 0;
  for (let i = 0; i < nums.length - 1; i++) {
    farthest = Math.max(farthest, i + nums[i]);
    if (i === end) {
      steps++;
      end = farthest;
    }
  }
  return steps;
}
```

## 96. 合并区间

- **标签**：排序 / 贪心
- **复杂度**：时间 O(n log n)，空间 O(n)

### 思路

先按左端点排序，再看当前区间是否能和结果数组最后一个区间合并。

### 代码（JavaScript）

```js
function merge(intervals) {
  intervals.sort((a, b) => a[0] - b[0]);
  const res = [];
  for (const cur of intervals) {
    if (!res.length || res[res.length - 1][1] < cur[0]) res.push(cur);
    else res[res.length - 1][1] = Math.max(res[res.length - 1][1], cur[1]);
  }
  return res;
}
```

## 97. 划分字母区间

- **标签**：贪心
- **复杂度**：时间 O(n)，空间 O(字符集大小)

### 思路

先统计每个字符最后出现位置，扫描时维护当前区间必须覆盖到的最远边界。

### 代码（JavaScript）

```js
function partitionLabels(s) {
  const last = {};
  for (let i = 0; i < s.length; i++) last[s[i]] = i;
  const res = [];
  let start = 0, end = 0;
  for (let i = 0; i < s.length; i++) {
    end = Math.max(end, last[s[i]]);
    if (i === end) {
      res.push(end - start + 1);
      start = i + 1;
    }
  }
  return res;
}
```

## 98. 任务调度器

- **标签**：贪心 / 计数
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

最高频任务决定最短框架长度，再和任务总数取最大值。

### 代码（JavaScript）

```js
function leastInterval(tasks, n) {
  const count = new Array(26).fill(0);
  for (const task of tasks) count[task.charCodeAt(0) - 65]++;
  count.sort((a, b) => b - a);
  const maxCount = count[0];
  let same = 1;
  while (same < 26 && count[same] === maxCount) same++;
  return Math.max(tasks.length, (maxCount - 1) * (n + 1) + same);
}
```

## 99. 最小栈 / 数据流结构题

- **标签**：设计
- **复杂度**：各操作 O(1)，空间 O(n)

### 思路

这类题重点是先定义操作复杂度，再选辅助结构。这里给一个最小栈模板。

### 代码（JavaScript）

```js
class MinStack {
  constructor() {
    this.stack = [];
    this.minStack = [];
  }
  push(val) {
    this.stack.push(val);
    this.minStack.push(this.minStack.length ? Math.min(val, this.minStack[this.minStack.length - 1]) : val);
  }
  pop() { this.minStack.pop(); return this.stack.pop(); }
  top() { return this.stack[this.stack.length - 1]; }
  getMin() { return this.minStack[this.minStack.length - 1]; }
}
```

## 100. LRU 缓存

- **标签**：设计 / 哈希表 / 双向链表
- **复杂度**：get/put 平均 O(1)，空间 O(capacity)

### 思路

哈希表负责 O(1) 定位节点，双向链表负责 O(1) 维护最近使用顺序。

### 代码（JavaScript）

```js
class Node {
  constructor(key, val) {
    this.key = key;
    this.val = val;
    this.prev = null;
    this.next = null;
  }
}
class LRUCache {
  constructor(capacity) {
    this.capacity = capacity;
    this.map = new Map();
    this.head = new Node();
    this.tail = new Node();
    this.head.next = this.tail;
    this.tail.prev = this.head;
  }
  remove(node) {
    node.prev.next = node.next;
    node.next.prev = node.prev;
  }
  insert(node) {
    node.next = this.head.next;
    node.prev = this.head;
    this.head.next.prev = node;
    this.head.next = node;
  }
  get(key) {
    if (!this.map.has(key)) return -1;
    const node = this.map.get(key);
    this.remove(node);
    this.insert(node);
    return node.val;
  }
  put(key, value) {
    if (this.map.has(key)) this.remove(this.map.get(key));
    const node = new Node(key, value);
    this.insert(node);
    this.map.set(key, node);
    if (this.map.size > this.capacity) {
      const lru = this.tail.prev;
      this.remove(lru);
      this.map.delete(lru.key);
    }
  }
}
```
