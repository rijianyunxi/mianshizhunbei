# 01 - 数组与哈希

数组与哈希是 Hot100 的起点。很多题本质上都在考 **如何用额外结构换时间**，或者 **如何利用数组下标本身做映射**。

## 这一类题怎么识别

- 题目要求快速判断“某个值是否出现过”
- 需要在一次遍历中同时完成查询和记录
- 数组元素范围和下标之间能建立映射

## 这类题的通用思路

优先想 `Map / Set / 前缀和 / 原地哈希` 四类解法。面试里最常见的是“边遍历边查找”的单遍哈希写法。

## 1. 两数之和

- **标签**：哈希表 / 数组
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

用 Map 记录已经遍历过的数字及其下标。当前数字只要能找到 `target - nums[i]`，说明答案已经出现。

### 代码（JavaScript）

```js
function twoSum(nums, target) {
  const map = new Map();
  for (let i = 0; i < nums.length; i++) {
    const need = target - nums[i];
    if (map.has(need)) return [map.get(need), i];
    map.set(nums[i], i);
  }
  return [];
}
```

## 2. 字母异位词分组

- **标签**：哈希表 / 字符串
- **复杂度**：时间 O(n * k log k)，空间 O(n)

### 思路

把排序后的字符串作为 key，相同 key 的字符串归到一组。

### 代码（JavaScript）

```js
function groupAnagrams(strs) {
  const map = new Map();
  for (const str of strs) {
    const key = str.split('').sort().join('');
    if (!map.has(key)) map.set(key, []);
    map.get(key).push(str);
  }
  return [...map.values()];
}
```

## 3. 最长连续序列

- **标签**：哈希表
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

先把所有数放进 Set，只从“连续段起点”开始向后扩展，避免重复统计。

### 代码（JavaScript）

```js
function longestConsecutive(nums) {
  const set = new Set(nums);
  let ans = 0;
  for (const num of set) {
    if (set.has(num - 1)) continue;
    let cur = num;
    let len = 1;
    while (set.has(cur + 1)) {
      cur++;
      len++;
    }
    ans = Math.max(ans, len);
  }
  return ans;
}
```

## 4. 多数元素

- **标签**：数组 / 投票算法
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

用 Boyer-Moore 投票法维护候选人。相同加一票，不同就相互抵消。

### 代码（JavaScript）

```js
function majorityElement(nums) {
  let candidate = null;
  let count = 0;
  for (const num of nums) {
    if (count === 0) candidate = num;
    count += num === candidate ? 1 : -1;
  }
  return candidate;
}
```

## 5. 除自身以外数组的乘积

- **标签**：数组 / 前后缀
- **复杂度**：时间 O(n)，空间 O(1)（不算结果数组）

### 思路

第一遍记录左侧乘积，第二遍从右向左乘上右侧乘积。

### 代码（JavaScript）

```js
function productExceptSelf(nums) {
  const res = new Array(nums.length).fill(1);
  let prefix = 1;
  for (let i = 0; i < nums.length; i++) {
    res[i] = prefix;
    prefix *= nums[i];
  }
  let suffix = 1;
  for (let i = nums.length - 1; i >= 0; i--) {
    res[i] *= suffix;
    suffix *= nums[i];
  }
  return res;
}
```

## 6. 缺失的第一个正数

- **标签**：数组 / 原地哈希
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

理想状态下值 `x` 应放在下标 `x - 1`。不断交换后，首个不匹配的位置就是答案。

### 代码（JavaScript）

```js
function firstMissingPositive(nums) {
  const n = nums.length;
  for (let i = 0; i < n; i++) {
    while (nums[i] >= 1 && nums[i] <= n && nums[nums[i] - 1] !== nums[i]) {
      const idx = nums[i] - 1;
      [nums[i], nums[idx]] = [nums[idx], nums[i]];
    }
  }
  for (let i = 0; i < n; i++) {
    if (nums[i] !== i + 1) return i + 1;
  }
  return n + 1;
}
```

## 7. 找到所有数组中消失的数字

- **标签**：数组 / 原地标记
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

利用元素值与下标的对应关系，把出现过的位置标记成负数。

### 代码（JavaScript）

```js
function findDisappearedNumbers(nums) {
  for (let i = 0; i < nums.length; i++) {
    const idx = Math.abs(nums[i]) - 1;
    if (nums[idx] > 0) nums[idx] = -nums[idx];
  }
  const res = [];
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] > 0) res.push(i + 1);
  }
  return res;
}
```

## 8. 子数组和等于 K

- **标签**：前缀和 / 哈希表
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

扫描时维护前缀和 `sum`。如果之前出现过 `sum - k`，就说明这段区间和为 `k`。

### 代码（JavaScript）

```js
function subarraySum(nums, k) {
  const map = new Map([[0, 1]]);
  let sum = 0;
  let ans = 0;
  for (const num of nums) {
    sum += num;
    ans += map.get(sum - k) || 0;
    map.set(sum, (map.get(sum) || 0) + 1);
  }
  return ans;
}
```

## 9. 存在重复元素

- **标签**：哈希表
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

遍历时把元素放入 Set，若已经存在则直接返回 true。

### 代码（JavaScript）

```js
function containsDuplicate(nums) {
  const set = new Set();
  for (const num of nums) {
    if (set.has(num)) return true;
    set.add(num);
  }
  return false;
}
```

## 10. 前 K 个高频元素

- **标签**：哈希表 / 排序
- **复杂度**：时间 O(n log n)，空间 O(n)

### 思路

先统计词频，再按频次排序取前 k 个，面试中 JS 这么写最直观。

### 代码（JavaScript）

```js
function topKFrequent(nums, k) {
  const map = new Map();
  for (const num of nums) {
    map.set(num, (map.get(num) || 0) + 1);
  }
  return [...map.entries()]
    .sort((a, b) => b[1] - a[1])
    .slice(0, k)
    .map(([num]) => num);
}
```
