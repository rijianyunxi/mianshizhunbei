# 10 - 动态规划进阶与贪心（10 题）

这篇里的题更像面试压轴题。不要只背答案，要能把“为什么这样定义状态、为什么贪心成立”讲明白。

## 91. 分割等和子集

- **标签**：动态规划 / 0/1 背包

### 思路

问题等价于能否选出一些数，使它们的和正好等于总和的一半。

### 代码（JavaScript）

```js
function canPartition(nums) {
  const sum = nums.reduce((a, b) => a + b, 0);
  if (sum % 2) return false;
  const target = sum / 2;
  const dp = new Array(target + 1).fill(false);
  dp[0] = true;
  for (const num of nums) {
    for (let j = target; j >= num; j--) {
      dp[j] = dp[j] || dp[j - num];
    }
  }
  return dp[target];
}
```

## 92. 最长有效括号

- **标签**：动态规划

### 思路

当当前位置是右括号时，尝试和前一个有效段或对应左括号拼接。

### 代码（JavaScript）

```js
function longestValidParentheses(s) {
  const dp = new Array(s.length).fill(0);
  let ans = 0;
  for (let i = 1; i < s.length; i++) {
    if (s[i] === ')') {
      if (s[i - 1] === '(') dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2;
      else {
        const pre = i - dp[i - 1] - 1;
        if (pre >= 0 && s[pre] === '(') {
          dp[i] = dp[i - 1] + 2 + (pre >= 1 ? dp[pre - 1] : 0);
        }
      }
      ans = Math.max(ans, dp[i]);
    }
  }
  return ans;
}
```

## 93. 最长公共子序列

- **标签**：动态规划

### 思路

字符相等就继承左上角加一，不相等就取左边和上边较大值。

### 代码（JavaScript）

```js
function longestCommonSubsequence(text1, text2) {
  const m = text1.length, n = text2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i - 1] === text2[j - 1]) dp[i][j] = dp[i - 1][j - 1] + 1;
      else dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
    }
  }
  return dp[m][n];
}
```

## 94. 编辑距离

- **标签**：动态规划

### 思路

当前字符不同时，在插入、删除、替换三种操作里取最小代价。

### 代码（JavaScript）

```js
function minDistance(word1, word2) {
  const m = word1.length, n = word2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));
  for (let i = 0; i <= m; i++) dp[i][0] = i;
  for (let j = 0; j <= n; j++) dp[0][j] = j;
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (word1[i - 1] === word2[j - 1]) dp[i][j] = dp[i - 1][j - 1];
      else dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) + 1;
    }
  }
  return dp[m][n];
}
```

## 95. 最长回文子串

- **标签**：动态规划 / 字符串

### 思路

定义 `dp[i][j]` 表示区间是否回文，按区间长度从短到长转移。

### 代码（JavaScript）

```js
function longestPalindrome(s) {
  const n = s.length;
  const dp = Array.from({ length: n }, () => new Array(n).fill(false));
  let start = 0, maxLen = 1;
  for (let i = n - 1; i >= 0; i--) {
    for (let j = i; j < n; j++) {
      if (s[i] === s[j] && (j - i <= 1 || dp[i + 1][j - 1])) {
        dp[i][j] = true;
        if (j - i + 1 > maxLen) {
          start = i;
          maxLen = j - i + 1;
        }
      }
    }
  }
  return s.slice(start, start + maxLen);
}
```

## 96. 回文子串

- **标签**：动态规划 / 中心扩散

### 思路

枚举每个中心向两边扩展，统计所有能扩出的回文。

### 代码（JavaScript）

```js
function countSubstrings(s) {
  let ans = 0;
  function expand(l, r) {
    while (l >= 0 && r < s.length && s[l] === s[r]) {
      ans++;
      l--;
      r++;
    }
  }
  for (let i = 0; i < s.length; i++) {
    expand(i, i);
    expand(i, i + 1);
  }
  return ans;
}
```

## 97. 买卖股票的最佳时机

- **标签**：贪心

### 思路

遍历时维护历史最低价，并用当前价减最低价更新最大利润。

### 代码（JavaScript）

```js
function maxProfit(prices) {
  let minPrice = Infinity, ans = 0;
  for (const price of prices) {
    minPrice = Math.min(minPrice, price);
    ans = Math.max(ans, price - minPrice);
  }
  return ans;
}
```

## 98. 跳跃游戏

- **标签**：贪心

### 思路

维护当前最远可达位置，只要遍历过程中没有断层，就能到终点。

### 代码（JavaScript）

```js
function canJump(nums) {
  let farthest = 0;
  for (let i = 0; i < nums.length; i++) {
    if (i > farthest) return false;
    farthest = Math.max(farthest, i + nums[i]);
  }
  return true;
}
```

## 99. 跳跃游戏 II

- **标签**：贪心

### 思路

按层推进可达范围，当走到当前层边界时步数加一并更新下一层边界。

### 代码（JavaScript）

```js
function jump(nums) {
  let end = 0, farthest = 0, steps = 0;
  for (let i = 0; i < nums.length - 1; i++) {
    farthest = Math.max(farthest, i + nums[i]);
    if (i === end) {
      steps++;
      end = farthest;
    }
  }
  return steps;
}
```

## 100. LRU 缓存

- **标签**：设计 / 哈希表 / 双向链表

### 思路

哈希表负责 O(1) 找节点，双向链表负责 O(1) 维护最近使用顺序。

### 代码（JavaScript）

```js
class Node {
  constructor(key, val) {
    this.key = key;
    this.val = val;
    this.prev = null;
    this.next = null;
  }
}
class LRUCache {
  constructor(capacity) {
    this.capacity = capacity;
    this.map = new Map();
    this.head = new Node();
    this.tail = new Node();
    this.head.next = this.tail;
    this.tail.prev = this.head;
  }
  remove(node) {
    node.prev.next = node.next;
    node.next.prev = node.prev;
  }
  insert(node) {
    node.next = this.head.next;
    node.prev = this.head;
    this.head.next.prev = node;
    this.head.next = node;
  }
  get(key) {
    if (!this.map.has(key)) return -1;
    const node = this.map.get(key);
    this.remove(node);
    this.insert(node);
    return node.val;
  }
  put(key, value) {
    if (this.map.has(key)) this.remove(this.map.get(key));
    const node = new Node(key, value);
    this.insert(node);
    this.map.set(key, node);
    if (this.map.size > this.capacity) {
      const lru = this.tail.prev;
      this.remove(lru);
      this.map.delete(lru.key);
    }
  }
}
```
