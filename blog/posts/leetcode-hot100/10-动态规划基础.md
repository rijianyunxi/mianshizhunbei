# 10 - 动态规划基础

动态规划最难的不是写代码，而是定义状态。状态定义清楚了，题已经做完一半。

## 这一类题怎么识别

- 当前答案依赖更小规模子问题
- 问题具有重复子问题和最优子结构
- 常见是“最值 / 方案数 / 是否可达”

## 这类题的通用思路

把 dp 含义说清楚，再写转移。面试里宁可慢一点，也别一上来就堆代码。

## 75. 爬楼梯

- **标签**：动态规划
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

到第 i 阶的方法数来自前两阶。

### 代码（JavaScript）

```js
function climbStairs(n) {
  if (n <= 2) return n;
  let a = 1, b = 2;
  for (let i = 3; i <= n; i++) [a, b] = [b, a + b];
  return b;
}
```

## 76. 杨辉三角

- **标签**：动态规划
- **复杂度**：时间 O(numRows^2)，空间 O(numRows^2)

### 思路

当前位置来自上一行左上角和右上角的和。

### 代码（JavaScript）

```js
function generate(numRows) {
  const res = [];
  for (let i = 0; i < numRows; i++) {
    const row = new Array(i + 1).fill(1);
    for (let j = 1; j < i; j++) row[j] = res[i - 1][j - 1] + res[i - 1][j];
    res.push(row);
  }
  return res;
}
```

## 77. 打家劫舍

- **标签**：动态规划
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

当前房屋要么不抢，要么抢了就加上前两间的最优值。

### 代码（JavaScript）

```js
function rob(nums) {
  let prev2 = 0, prev1 = 0;
  for (const num of nums) {
    const cur = Math.max(prev1, prev2 + num);
    prev2 = prev1;
    prev1 = cur;
  }
  return prev1;
}
```

## 78. 完全平方数

- **标签**：动态规划
- **复杂度**：时间 O(n*sqrt(n))，空间 O(n)

### 思路

枚举最后一个选用的平方数，转移到更小状态。

### 代码（JavaScript）

```js
function numSquares(n) {
  const dp = new Array(n + 1).fill(Infinity);
  dp[0] = 0;
  for (let i = 1; i <= n; i++) {
    for (let j = 1; j * j <= i; j++) {
      dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
    }
  }
  return dp[n];
}
```

## 79. 零钱兑换

- **标签**：动态规划
- **复杂度**：时间 O(amount * coins.length)，空间 O(amount)

### 思路

`dp[i]` 表示凑成金额 i 的最少硬币数。

### 代码（JavaScript）

```js
function coinChange(coins, amount) {
  const dp = new Array(amount + 1).fill(Infinity);
  dp[0] = 0;
  for (let i = 1; i <= amount; i++) {
    for (const coin of coins) {
      if (i >= coin) dp[i] = Math.min(dp[i], dp[i - coin] + 1);
    }
  }
  return dp[amount] === Infinity ? -1 : dp[amount];
}
```

## 80. 单词拆分

- **标签**：动态规划
- **复杂度**：时间 O(n^2)，空间 O(n)

### 思路

`dp[i]` 表示前 i 个字符能否由字典拆分出来。

### 代码（JavaScript）

```js
function wordBreak(s, wordDict) {
  const set = new Set(wordDict);
  const dp = new Array(s.length + 1).fill(false);
  dp[0] = true;
  for (let i = 1; i <= s.length; i++) {
    for (let j = 0; j < i; j++) {
      if (dp[j] && set.has(s.slice(j, i))) {
        dp[i] = true;
        break;
      }
    }
  }
  return dp[s.length];
}
```

## 81. 最长递增子序列

- **标签**：动态规划
- **复杂度**：时间 O(n^2)，空间 O(n)

### 思路

`dp[i]` 是以 `nums[i]` 结尾的 LIS 长度。

### 代码（JavaScript）

```js
function lengthOfLIS(nums) {
  const dp = new Array(nums.length).fill(1);
  let ans = 1;
  for (let i = 1; i < nums.length; i++) {
    for (let j = 0; j < i; j++) {
      if (nums[i] > nums[j]) dp[i] = Math.max(dp[i], dp[j] + 1);
    }
    ans = Math.max(ans, dp[i]);
  }
  return ans;
}
```

## 82. 乘积最大子数组

- **标签**：动态规划
- **复杂度**：时间 O(n)，空间 O(1)

### 思路

负数会让最大最小互换，所以要同时维护 max 和 min。

### 代码（JavaScript）

```js
function maxProduct(nums) {
  let maxF = nums[0], minF = nums[0], ans = nums[0];
  for (let i = 1; i < nums.length; i++) {
    const x = nums[i];
    const candidates = [x, maxF * x, minF * x];
    maxF = Math.max(...candidates);
    minF = Math.min(...candidates);
    ans = Math.max(ans, maxF);
  }
  return ans;
}
```

## 83. 不同路径

- **标签**：动态规划
- **复杂度**：时间 O(mn)，空间 O(mn)

### 思路

到达当前位置的路径数等于上方和左侧路径数之和。

### 代码（JavaScript）

```js
function uniquePaths(m, n) {
  const dp = Array.from({ length: m }, () => new Array(n).fill(1));
  for (let i = 1; i < m; i++) {
    for (let j = 1; j < n; j++) {
      dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
    }
  }
  return dp[m - 1][n - 1];
}
```

## 84. 最小路径和

- **标签**：动态规划
- **复杂度**：时间 O(mn)，空间 O(mn)

### 思路

当前位置最优解来自上边和左边较小的那个。

### 代码（JavaScript）

```js
function minPathSum(grid) {
  const m = grid.length, n = grid[0].length;
  const dp = Array.from({ length: m }, () => new Array(n).fill(0));
  dp[0][0] = grid[0][0];
  for (let i = 1; i < m; i++) dp[i][0] = dp[i - 1][0] + grid[i][0];
  for (let j = 1; j < n; j++) dp[0][j] = dp[0][j - 1] + grid[0][j];
  for (let i = 1; i < m; i++) {
    for (let j = 1; j < n; j++) {
      dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
    }
  }
  return dp[m - 1][n - 1];
}
```
