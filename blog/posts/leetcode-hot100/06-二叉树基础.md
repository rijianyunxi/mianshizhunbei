# 06 - 二叉树基础

树题写不出来，通常不是代码不熟，而是没有定义清楚递归函数到底返回什么。

## 这一类题怎么识别

- 题目围绕深度、遍历、路径、层序展开
- 子问题天然是“左子树 / 右子树”
- 答案可以由左右子树结果组合出来

## 这类题的通用思路

先问：我的递归函数是“遍历节点做事”，还是“返回以当前节点为根的某个答案”？

## 43. 二叉树的中序遍历

- **标签**：二叉树 / DFS
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

中序遍历顺序是左根右，递归是最直接写法。

### 代码（JavaScript）

```js
function inorderTraversal(root) {
  const res = [];
  function dfs(node) {
    if (!node) return;
    dfs(node.left);
    res.push(node.val);
    dfs(node.right);
  }
  dfs(root);
  return res;
}
```

## 44. 二叉树的最大深度

- **标签**：二叉树 / DFS
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

当前节点最大深度等于左右子树最大深度中的较大值加一。

### 代码（JavaScript）

```js
function maxDepth(root) {
  if (!root) return 0;
  return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
}
```

## 45. 翻转二叉树

- **标签**：二叉树
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

交换每个节点的左右子树即可。

### 代码（JavaScript）

```js
function invertTree(root) {
  if (!root) return null;
  [root.left, root.right] = [root.right, root.left];
  invertTree(root.left);
  invertTree(root.right);
  return root;
}
```

## 46. 对称二叉树

- **标签**：二叉树
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

比较左树外侧和右树外侧、左树内侧和右树内侧是否相等。

### 代码（JavaScript）

```js
function isSymmetric(root) {
  function check(a, b) {
    if (!a && !b) return true;
    if (!a || !b || a.val !== b.val) return false;
    return check(a.left, b.right) && check(a.right, b.left);
  }
  return check(root.left, root.right);
}
```

## 47. 二叉树的直径

- **标签**：二叉树 / DFS
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

以每个节点为拐点，尝试用左右深度之和更新直径。

### 代码（JavaScript）

```js
function diameterOfBinaryTree(root) {
  let ans = 0;
  function depth(node) {
    if (!node) return 0;
    const left = depth(node.left);
    const right = depth(node.right);
    ans = Math.max(ans, left + right);
    return Math.max(left, right) + 1;
  }
  depth(root);
  return ans;
}
```

## 48. 二叉树的层序遍历

- **标签**：二叉树 / BFS
- **复杂度**：时间 O(n)，空间 O(n)

### 思路

用队列按层处理，每轮先记录当前层节点数。

### 代码（JavaScript）

```js
function levelOrder(root) {
  if (!root) return [];
  const queue = [root];
  const res = [];
  while (queue.length) {
    const size = queue.length;
    const level = [];
    for (let i = 0; i < size; i++) {
      const node = queue.shift();
      level.push(node.val);
      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }
    res.push(level);
  }
  return res;
}
```

## 49. 将有序数组转换为二叉搜索树

- **标签**：BST / 分治
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

每次取中点做根节点，左右两边递归构造平衡 BST。

### 代码（JavaScript）

```js
function sortedArrayToBST(nums) {
  function build(l, r) {
    if (l > r) return null;
    const mid = Math.floor((l + r) / 2);
    const root = new TreeNode(nums[mid]);
    root.left = build(l, mid - 1);
    root.right = build(mid + 1, r);
    return root;
  }
  return build(0, nums.length - 1);
}
```

## 50. 验证二叉搜索树

- **标签**：BST
- **复杂度**：时间 O(n)，空间 O(h)

### 思路

BST 判断不能只比父节点，而要维护全局合法区间。

### 代码（JavaScript）

```js
function isValidBST(root) {
  function dfs(node, min, max) {
    if (!node) return true;
    if (node.val <= min || node.val >= max) return false;
    return dfs(node.left, min, node.val) && dfs(node.right, node.val, max);
  }
  return dfs(root, -Infinity, Infinity);
}
```
