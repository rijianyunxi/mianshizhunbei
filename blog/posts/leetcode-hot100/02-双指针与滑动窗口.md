# 02 - 双指针与滑动窗口（10 题）

双指针和滑动窗口的共同点是：**都在利用区间信息做增量更新，而不是每次都从头算一遍。**

## 11. 盛最多水的容器

- **标签**：双指针

### 思路

左右指针夹逼，面积由较短板决定，所以每次移动较短的一侧。

### 代码（JavaScript）

```js
function maxArea(height) {
  let l = 0, r = height.length - 1;
  let ans = 0;
  while (l < r) {
    ans = Math.max(ans, Math.min(height[l], height[r]) * (r - l));
    if (height[l] < height[r]) l++;
    else r--;
  }
  return ans;
}
```

## 12. 三数之和

- **标签**：排序 / 双指针

### 思路

先排序，固定一个数，再在后面区间用双指针找另外两个数。

### 代码（JavaScript）

```js
function threeSum(nums) {
  nums.sort((a, b) => a - b);
  const res = [];
  for (let i = 0; i < nums.length - 2; i++) {
    if (i > 0 && nums[i] === nums[i - 1]) continue;
    let l = i + 1, r = nums.length - 1;
    while (l < r) {
      const sum = nums[i] + nums[l] + nums[r];
      if (sum === 0) {
        res.push([nums[i], nums[l], nums[r]]);
        while (l < r && nums[l] === nums[l + 1]) l++;
        while (l < r && nums[r] === nums[r - 1]) r--;
        l++;
        r--;
      } else if (sum < 0) l++;
      else r--;
    }
  }
  return res;
}
```

## 13. 移动零

- **标签**：双指针

### 思路

慢指针指向下一个非零元素应放的位置，快指针负责扫描。

### 代码（JavaScript）

```js
function moveZeroes(nums) {
  let slow = 0;
  for (let fast = 0; fast < nums.length; fast++) {
    if (nums[fast] !== 0) {
      [nums[slow], nums[fast]] = [nums[fast], nums[slow]];
      slow++;
    }
  }
  return nums;
}
```

## 14. 接雨水

- **标签**：双指针

### 思路

维护左右两边最高柱子，较低的一边可以直接结算当前能接多少水。

### 代码（JavaScript）

```js
function trap(height) {
  let l = 0, r = height.length - 1;
  let leftMax = 0, rightMax = 0, ans = 0;
  while (l < r) {
    leftMax = Math.max(leftMax, height[l]);
    rightMax = Math.max(rightMax, height[r]);
    if (leftMax < rightMax) {
      ans += leftMax - height[l];
      l++;
    } else {
      ans += rightMax - height[r];
      r--;
    }
  }
  return ans;
}
```

## 15. 无重复字符的最长子串

- **标签**：滑动窗口 / 哈希表

### 思路

窗口右扩时记录字符出现次数，若重复则不断左缩直到重新合法。

### 代码（JavaScript）

```js
function lengthOfLongestSubstring(s) {
  const map = new Map();
  let left = 0, ans = 0;
  for (let right = 0; right < s.length; right++) {
    const ch = s[right];
    map.set(ch, (map.get(ch) || 0) + 1);
    while (map.get(ch) > 1) {
      const leftChar = s[left++];
      map.set(leftChar, map.get(leftChar) - 1);
    }
    ans = Math.max(ans, right - left + 1);
  }
  return ans;
}
```

## 16. 找到字符串中所有字母异位词

- **标签**：滑动窗口

### 思路

维护一个固定长度窗口和目标字符串计数表，窗口满足时记录左边界。

### 代码（JavaScript）

```js
function findAnagrams(s, p) {
  const need = new Array(26).fill(0);
  const win = new Array(26).fill(0);
  for (const ch of p) need[ch.charCodeAt(0) - 97]++;
  const res = [];
  let left = 0;
  for (let right = 0; right < s.length; right++) {
    win[s.charCodeAt(right) - 97]++;
    if (right - left + 1 > p.length) {
      win[s.charCodeAt(left) - 97]--;
      left++;
    }
    if (right - left + 1 === p.length && need.join(',') === win.join(',')) {
      res.push(left);
    }
  }
  return res;
}
```

## 17. 最小覆盖子串

- **标签**：滑动窗口 / 哈希表

### 思路

先扩张窗口直到覆盖所有目标字符，再尽量收缩左边界更新最短答案。

### 代码（JavaScript）

```js
function minWindow(s, t) {
  const need = new Map();
  for (const ch of t) need.set(ch, (need.get(ch) || 0) + 1);
  let needType = need.size;
  let left = 0, start = 0, minLen = Infinity;
  for (let right = 0; right < s.length; right++) {
    const ch = s[right];
    if (need.has(ch)) {
      need.set(ch, need.get(ch) - 1);
      if (need.get(ch) === 0) needType--;
    }
    while (needType === 0) {
      if (right - left + 1 < minLen) {
        minLen = right - left + 1;
        start = left;
      }
      const leftChar = s[left++];
      if (need.has(leftChar)) {
        if (need.get(leftChar) === 0) needType++;
        need.set(leftChar, need.get(leftChar) + 1);
      }
    }
  }
  return minLen === Infinity ? '' : s.slice(start, start + minLen);
}
```

## 18. 滑动窗口最大值

- **标签**：滑动窗口 / 单调队列

### 思路

队列里存下标，并保持对应值单调递减，队头始终是当前窗口最大值。

### 代码（JavaScript）

```js
function maxSlidingWindow(nums, k) {
  const deque = [];
  const res = [];
  for (let i = 0; i < nums.length; i++) {
    while (deque.length && nums[deque[deque.length - 1]] <= nums[i]) deque.pop();
    deque.push(i);
    if (deque[0] <= i - k) deque.shift();
    if (i >= k - 1) res.push(nums[deque[0]]);
  }
  return res;
}
```

## 19. 长度最小的子数组

- **标签**：滑动窗口

### 思路

当窗口和已经大于等于目标值，就不断缩左边界以更新最短长度。

### 代码（JavaScript）

```js
function minSubArrayLen(target, nums) {
  let left = 0, sum = 0, ans = Infinity;
  for (let right = 0; right < nums.length; right++) {
    sum += nums[right];
    while (sum >= target) {
      ans = Math.min(ans, right - left + 1);
      sum -= nums[left++];
    }
  }
  return ans === Infinity ? 0 : ans;
}
```

## 20. 乘积小于 K 的子数组

- **标签**：滑动窗口

### 思路

窗口内维护乘积，超过上限就左移。每次窗口长度都代表以 right 结尾的新子数组数。

### 代码（JavaScript）

```js
function numSubarrayProductLessThanK(nums, k) {
  if (k <= 1) return 0;
  let left = 0, product = 1, ans = 0;
  for (let right = 0; right < nums.length; right++) {
    product *= nums[right];
    while (product >= k) product /= nums[left++];
    ans += right - left + 1;
  }
  return ans;
}
```
