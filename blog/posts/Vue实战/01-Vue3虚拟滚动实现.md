# Vue3 虚拟滚动实现

Vue3 中实现虚拟滚动，原理与 React 类似，但语法不同。

## 1. 基础实现

```vue
<template>
  <div class="virtual-list" @scroll="onScroll">
    <div class="spacer" :style="{ height: totalHeight + 'px' }">
      <div
        class="visible-items"
        :style="{ transform: `translateY(${offsetY}px)` }"
      >
        <div
          v-for="item in visibleItems"
          :key="item.id"
          class="item"
        >
          {{ item.content }}
        </div>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const props = defineProps({
  items: Array,
  itemHeight: {
    type: Number,
    default: 50
  }
})

const scrollTop = ref(0)
const buffer = 3

const totalHeight = computed(() => props.items.length * props.itemHeight)

const startIndex = computed(() => 
  Math.floor(scrollTop.value / props.itemHeight)
)

const endIndex = computed(() => 
  Math.min(
    props.items.length,
    Math.ceil((scrollTop.value + 400) / props.itemHeight)
  )
)

const visibleItems = computed(() => 
  props.items.slice(
    Math.max(0, startIndex.value - buffer),
    endIndex.value + buffer
  )
)

const offsetY = computed(() => 
  Math.max(0, startIndex.value - buffer) * props.itemHeight
)

const onScroll = (e) => {
  scrollTop.value = e.target.scrollTop
}
</script>
```

## 3. 动态高度

```vue
<template>
  <div ref="container" @scroll="onScroll">
    <div :style="{ height: totalHeight + 'px', position: 'relative' }">
      <div
        v-for="item in visibleItems"
        :key="item.id"
        :style="{ position: 'absolute', top: getOffset(item) + 'px', width: '100%' }"
        ref="itemRefs"
      >
        {{ item.content }}
      </div>
    </div>
  </div>
</template>

<script setup>
const heights = new Map()
const estimatedHeight = 50

const getOffset = (item) => {
  let offset = 0
  for (const i of props.items) {
    if (i.id === item.id) break
    offset += heights.get(i.id) || estimatedHeight
  }
  return offset
}
</script>
```

## 5. 面试高频问题

### Q: Vue 和 React 虚拟滚动实现的异同？

- 原理相同：只渲染可视区域元素
- 语法不同：Vue 用模板，React 用 JSX
- 响应式不同：Vue 自动追踪，React 手动优化

### Q: 动态高度如何处理？

预估高度 + 测量实际高度 + 缓存。

## 6. 总结

Vue3 虚拟滚动核心：
- 计算可视区域索引
- 只渲染可见元素
- transform 定位避免重排
