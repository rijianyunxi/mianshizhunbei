# 压缩字符串与反转字符串

字符串操作是算法面试中的基础题型，本文讲解常见面试题。

## 1. 字符串压缩

### 题目：连续字符计数压缩

```javascript
function compress(str) {
  let result = ''
  let count = 1
  
  for (let i = 1; i <= str.length; i++) {
    if (str[i] === str[i - 1]) {
      count++
    } else {
      result += str[i - 1] + (count > 1 ? count : '')
      count = 1
    }
  }
  
  return result
}

compress('aabcccccaaa') // 'a2b1c5a3'
compress('abcdef') // 'abcdef'
```

## 2. 反转字符串

### 方法1：双指针

```javascript
function reverseString(s) {
  const arr = s.split('')
  let left = 0
  let right = arr.length - 1
  
  while (left < right) {
    [arr[left], arr[right]] = [arr[right], arr[left]]
    left++
    right--
  }
  
  return arr.join('')
}

reverseString('hello') // 'olleh'
```

### 方法2：reverse

```javascript
function reverseString(s) {
  return s.split('').reverse().join('')
}
```

### 方法3：递归

```javascript
function reverseString(s) {
  if (s.length <= 1) return s
  return reverseString(s.slice(1)) + s[0]
}
```

## 3. 单词反转

```javascript
function reverseWords(str) {
  return str.split(' ').reverse().join(' ')
}

reverseWords('hello world') // 'world hello'
```

## 4. 左旋转字符串

```javascript
function leftRotate(str, n) {
  return str.slice(n) + str.slice(0, n)
}

leftRotate('abcdefg', 2) // 'cdefgab'
```

## 5. 双指针技巧

### 应用场景

- 两数之和
- 反转链表
- 滑动窗口
- 删除重复项

```javascript
// 删除排序数组中的重复项
function removeDuplicates(nums) {
  if (nums.length === 0) return 0
  
  let slow = 0
  for (let fast = 1; fast < nums.length; fast++) {
    if (nums[fast] !== nums[slow]) {
      slow++
      nums[slow] = nums[fast]
    }
  }
  
  return slow + 1
}
```

## 6. 面试高频问题

### Q: 双指针的时间复杂度？

O(n)，通常比暴力解法 O(n²) 更快。

### Q: 什么时候用双指针？

- 两数之和（左右指针）
- 反转链表（快慢指针）
- 滑动窗口（左右指针）

## 7. 总结

字符串操作的核心技巧：
- 双指针：反转、两数之和
- 计数：压缩字符串
- 分割：单词反转
