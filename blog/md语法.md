# Markdown 快速上手指南

这是一份基础的 Markdown 语法指南，非常适合用于编写文档、博客或日常笔记。

## 1. 标题 (Headings)

使用 `#` 号可以表示 1-6 级标题，`#` 和标题文字之间需要保留一个空格。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 2. 文本格式 (Text Formatting)

**加粗文本**：在文字两端加上两个星号 `**`
*斜体文本*：在文字两端加上一个星号 `*`
***加粗斜体***：在文字两端加上三个星号 `***`
~~删除线~~：在文字两端加上两个波浪号 `~~`

## 3. 列表 (Lists)

### 无序列表
使用 `-`、`*` 或 `+` 加空格表示无序列表：

- 苹果
- 香蕉
  - 嵌套列表项 (前面加两个空格缩进)
- 橘子

### 有序列表
使用数字加英文句号表示有序列表：

1. 打开冰箱
2. 把大象放进去
3. 关上冰箱

### 任务列表 (Task Lists)
- [x] 已完成的任务 (中括号内写小写 x)
- [ ] 待办事项 1 (中括号内留一个空格)
- [ ] 待办事项 2

## 4. 链接与图片 (Links & Images)

**超链接**：`[显示的文字](链接地址)`
[点击访问百度](https://www.baidu.com)

**图片**：`![图片加载失败时的替代文字](图片网络地址或本地路径)`
![示例图片](https://picsum.photos/200/300)

## 5. 引用 (Blockquotes)

使用 `>` 加空格表示引用段落：

> 这是一个引用段落。
> 可以包含多行内容。
> > 甚至可以嵌套引用！

## 6. 代码 (Code)

**行内代码**：使用反引号（键盘 Esc 下方的按键）包裹，例如 `npm run dev`。

**代码块**：使用三个反引号包裹多行代码，并在第一行末尾指定语言（如 js, bash, python, vue）以开启语法高亮。

```javascript
// 这是一个 JavaScript 代码块示例
function greet(name) {
  console.log(`Hello, ${name}!`);
}
greet('VitePress');
```

## 7. 表格 (Tables)

使用 `|` 分隔不同的列，使用 `-` 分隔表头和内容行。可以通过冒号 `:` 控制对齐方式（可选）。

| 默认左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 单元格 1 | 单元格 2 | 单元格 3 |
| Apple | Banana | Orange |

## 8. 分隔线 (Horizontal Rules)

在一行中独立使用三个或更多的减号 `-`、星号 `*` 或下划线 `_` 即可生成一条水平分隔线。
